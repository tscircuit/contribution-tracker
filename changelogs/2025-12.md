# Changelog 2025-12

- Copper text everywhere – full rendering + knockout support added to pcb-viewer (#525), 3D-viewer (#617), SVG exporter (#442, #444) and Gerber generator (#74)
- PCB-viewer improvements – pill-pad rotation fix (#523), correct panel centering (#520), text baseline alignment (#515) and context-menu tweaks (right-click disabled #518, shape-handler refactor #517)
- 3D-viewer overhaul – solder-mask texturing & visibility controls (#595, #604-606, #614), fabrication-note rectangles (#597) and manifold trace/z-fighting fixes (#611, #618)
- Panelization refresh in core – new tab & mouse-bite engine (#1729), tab-routing for custom outlines (#1728), default “none” panelization (#1722) plus grid-packing refactor & lifecycle events (#1736, #1734)
- Packing accuracy – components treated as static when positioned relatively (#1730); calculate-packing gains vias-as-pads, inner-pad obstruction & weighted connections (#84, #83, #81); rectdiff adds board-outline support (#20)
- Autorouter upgrades – multilayer connection points, net-merging logic and iterative trace-simplification solver (#362, #364, #366) with new high-density & LGA examples (#387-390)
- SPICE fidelity – proper MOSFET models, fixed inductance handling and buck-converter test support (core #1727; circuit-json-to-spice #28-30)
- PCB JSON enhancements – polygon SMT-pad transforms (#71), source_component_internal_connection element (#368) and solder-mask flag fix for plated holes (#367)
- New panel props – <fiducial/>, <subpanel/>, extra layout modes & boardGap (props #526-527)
- CLI/Tooling – KiCad footprint-library export (cli #1254), better preview-image fallback (#1270), charset meta tags in generated pages (#1296) and clearer transpile warnings (#1290)
- GLTF pipeline – translucent model support & OBJ position fix (circuit-json-to-gltf #96, #95) with matching docs (#366)
- JSCAD & web fixes – board colors, cut-out rotation, solder-mask coverage for oval/plated holes (#598-606, #610)
- DSN import/export – full SpectraSession parsing and freerouting spec tests (dsn-to-circuit-json #1-9; circuit-json-to-dsn #6-10)
- Misc – via geometry improvements (#594), treat relatively positioned parts as static (#1730), add SPICE filter docs (#371) and new sponsorship logic in contribution-tracker (#274)
