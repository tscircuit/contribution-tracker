# Changelog 2026-07

- Interactive schematic clarity: hovering a net now fades all unrelated nets and chips ([schematic-viewer #237](https://github.com/tscircuit/schematic-viewer/pull/237)).
- Run multiple analog simulations and pick them in–app ([schematic-viewer #240](https://github.com/tscircuit/schematic-viewer/pull/240), [core #2677](https://github.com/tscircuit/core/pull/2677)).
- New `tsci check-shorts` command with bitmap short-detection and debug artifacts ([cli #3587](https://github.com/tscircuit/cli/pull/3587)).
- Autorouter upgraded to Pipeline 7 with smarter BGA handling, better via merging and safer stitch routing ([core #2618](https://github.com/tscircuit/core/pull/2618), [tscircuit-autorouter #1545](https://github.com/tscircuit/tscircuit-autorouter/pull/1545), [#1537](https://github.com/tscircuit/tscircuit-autorouter/pull/1537), [#1536](https://github.com/tscircuit/tscircuit-autorouter/pull/1536)).
- Differential-pair length-matching solver added and integrated into Pipeline 7 ([length-matching-solver #1](https://github.com/tscircuit/length-matching-solver/pull/1), [tscircuit-autorouter #1618](https://github.com/tscircuit/tscircuit-autorouter/pull/1618)).
- Flexible MOSFET symbols: new orientation variants and port-side props implemented ([schematic-symbols #437](https://github.com/tscircuit/schematic-symbols/pull/437), [props #726](https://github.com/tscircuit/props/pull/726), [core #2652](https://github.com/tscircuit/core/pull/2652)).
- Crystal safety: four-pin variants supported and warnings added for over-length traces ([props #729](https://github.com/tscircuit/props/pull/729), [core #2643](https://github.com/tscircuit/core/pull/2643)).
- Gerber output improvements: keep silkscreen text case and render `pcb_silkscreen_graphic` shapes ([circuit-json-to-gerber #120](https://github.com/tscircuit/circuit-json-to-gerber/pull/120), [#122](https://github.com/tscircuit/circuit-json-to-gerber/pull/122)).
- KiCad export fixes: subtract edge cutouts correctly and mirror bottom-layer silkscreen text ([circuit-json-to-kicad #365](https://github.com/tscircuit/circuit-json-to-kicad/pull/365), [#373](https://github.com/tscircuit/circuit-json-to-kicad/pull/373)).
- KiCad import reliability: legacy KiCad 5 footprints and rotated custom pad polygons now convert accurately ([kicad-to-circuit-json #160](https://github.com/tscircuit/kicad-to-circuit-json/pull/160), [#167](https://github.com/tscircuit/kicad-to-circuit-json/pull/167)).
- SVG accuracy: silkscreen pills rotate with `ccw_rotation` and collapsed group ports show terminal dots ([circuit-to-svg #617](https://github.com/tscircuit/circuit-to-svg/pull/617), [#621](https://github.com/tscircuit/circuit-to-svg/pull/621)).
- Silkscreen text placement & pin-label fixes in schematics ([core #2615](https://github.com/tscircuit/core/pull/2615), [schematic-trace-solver #654](https://github.com/tscircuit/schematic-trace-solver/pull/654)).
