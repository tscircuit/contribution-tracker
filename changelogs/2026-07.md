# Changelog 2026-07

- Schematic Viewer: hovering a net now highlights it while fading all others for quick visual isolation ([schematic-viewer #237](https://github.com/tscircuit/schematic-viewer/pull/237)).
- CLI: added `tsci check-shorts` bitmap short-detection command and optional routing-artifact images for autorouter debugging ([cli #3587](https://github.com/tscircuit/cli/pull/3587), [cli #3612](https://github.com/tscircuit/cli/pull/3612)).
- Autorouter Pipeline 7 is now the default and ships output-via counts, differential-pair support, and merged global/component topology colours ([core #2618](https://github.com/tscircuit/core/pull/2618), [autorouter #1556](https://github.com/tscircuit/tscircuit-autorouter/pull/1556), [autorouter #1583](https://github.com/tscircuit/tscircuit-autorouter/pull/1583), [autorouter #1592](https://github.com/tscircuit/tscircuit-autorouter/pull/1592)).
- Decoupling-capacitor layout: new Rail-Row solver and collision fixes with series resistors ([matchpack #155](https://github.com/tscircuit/matchpack/pull/155), [matchpack #153](https://github.com/tscircuit/matchpack/pull/153)).
- Schematic routing fixes: correct out-of-order pin labels, honour pin facing directions, provide ratsnest-free snapshots, and resolve recent routing regressions ([core #2615](https://github.com/tscircuit/core/pull/2615), [schematic-trace-solver #645](https://github.com/tscircuit/schematic-trace-solver/pull/645), [#646](https://github.com/tscircuit/schematic-trace-solver/pull/646), [#648](https://github.com/tscircuit/schematic-trace-solver/pull/648)).
- MOSFET symbol upgrades: new orientation variants, side-specific connection props and aligned terminals ([schematic-symbols #437](https://github.com/tscircuit/schematic-symbols/pull/437), [props #720](https://github.com/tscircuit/props/pull/720), [props #726](https://github.com/tscircuit/props/pull/726), [schematic-symbols #436](https://github.com/tscircuit/schematic-symbols/pull/436)).
- Gerber/KiCad exports: keep silkscreen text case, render graphics as filled regions, fix cutout subtraction & mirrored bottom-layer text, preserve text sizes, and support legacy KiCad 5 footprints ([circuit-json-to-gerber #120](https://github.com/tscircuit/circuit-json-to-gerber/pull/120), [#122](https://github.com/tscircuit/circuit-json-to-gerber/pull/122), [circuit-json-to-kicad #365](https://github.com/tscircuit/circuit-json-to-kicad/pull/365), [#373](https://github.com/tscircuit/circuit-json-to-kicad/pull/373), [#375](https://github.com/tscircuit/circuit-json-to-kicad/pull/375), [kicad-to-circuit-json #160](https://github.com/tscircuit/kicad-to-circuit-json/pull/160)).
- Check-Shorts: faster, more accurate bitmap short detection and first public package release ([check-shorts #7](https://github.com/tscircuit/check-shorts/pull/7), [#9](https://github.com/tscircuit/check-shorts/pull/9)).
- New design-rule warnings for over-length crystal traces and reserved refdes prefixes ([core #2643](https://github.com/tscircuit/core/pull/2643), [core #2640](https://github.com/tscircuit/core/pull/2640)).
- Circuit-to-SVG: PCB bounds now include holes/notes/cutouts; net-label elements are tagged; long schematic names stay within SVG ([circuit-to-svg #607](https://github.com/tscircuit/circuit-to-svg/pull/607), [#602](https://github.com/tscircuit/circuit-to-svg/pull/602), [#615](https://github.com/tscircuit/circuit-to-svg/pull/615)).
- Brand-new differential-pair length-matching solver with visualisation fixtures ([differential-pairs-solver #1](https://github.com/tscircuit/differential-pairs-solver/pull/1)).
- KiCad toolkit: public accessors for PCB dimensions ([kicadts #57](https://github.com/tscircuit/kicadts/pull/57)).
