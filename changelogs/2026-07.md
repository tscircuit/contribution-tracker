# Changelog 2026-07

- Hover a net in the schematic viewer to automatically fade unrelated nets and chips, making signal tracing easier ([schematic-viewer #237](https://github.com/tscircuit/schematic-viewer/pull/237))
- Added `tsci check-shorts` CLI command for fast bitmap-based PCB/Gerber short detection with debug images ([cli #3587](https://github.com/tscircuit/cli/pull/3587))
- Gerber exporter improvements:
  • keeps original silkscreen text case 
  • outputs `pcb_silkscreen_graphic` as filled regions 
  • prevents overlapping board cut-outs  
  ([circuit-json-to-gerber #120](https://github.com/tscircuit/circuit-json-to-gerber/pull/120), [#122](https://github.com/tscircuit/circuit-json-to-gerber/pull/122), [#123](https://github.com/tscircuit/circuit-json-to-gerber/pull/123))
- KiCad exporter fixes board outline/cut-out maths, mirrors bottom-layer silkscreen correctly and preserves text thickness & size ([circuit-json-to-kicad #365](https://github.com/tscircuit/circuit-json-to-kicad/pull/365), [#373](https://github.com/tscircuit/circuit-json-to-kicad/pull/373), [#375](https://github.com/tscircuit/circuit-json-to-kicad/pull/375))
- KiCad importer now supports legacy KiCad-5 standalone footprints and improves connectivity conversion ([kicad-to-circuit-json #160](https://github.com/tscircuit/kicad-to-circuit-json/pull/160), [#163](https://github.com/tscircuit/kicad-to-circuit-json/pull/163))
- Power & ground net labels now snap neatly to a pin-aligned corner in schematics ([schematic-trace-solver #654](https://github.com/tscircuit/schematic-trace-solver/pull/654))
- New ratsnest-free schematic solver visualizations for cleaner bug-report snapshots ([schematic-trace-solver #646](https://github.com/tscircuit/schematic-trace-solver/pull/646))
- Decoupling capacitors are now auto-placed in compact rail rows via the new DecouplingCapRowSolver ([matchpack #155](https://github.com/tscircuit/matchpack/pull/155))
- Autorouter correctly aligns BGA grids when routing around obstacles ([tscircuit-autorouter #1545](https://github.com/tscircuit/tscircuit-autorouter/pull/1545))
- MOSFET connection support added to props & switch/MOSFET symbol terminal alignment fixed for better schematic routing ([props #720](https://github.com/tscircuit/props/pull/720), [schematic-symbols #436](https://github.com/tscircuit/schematic-symbols/pull/436))
- Monaco code editor shows file-type icons (TSX, JSON, MD) in the workspace tree for quicker navigation ([monaco-code-editor #17](https://github.com/tscircuit/monaco-code-editor/pull/17))
- Generate “simulation-only” snippet URLs for lightweight sharing of SPICE results ([create-snippet-url #11](https://github.com/tscircuit/create-snippet-url/pull/11))
- Stand-alone simulation graph SVGs supported via svg.tscircuit.com, preventing Next.js router crashes ([svg.tscircuit.com #1763](https://github.com/tscircuit/svg.tscircuit.com/pull/1763))
- System-block-designer gains an embedded AI assistant chat for faster diagram creation ([system-block-designer #55](https://github.com/tscircuit/system-block-designer/pull/55))
- Bitmap-based short-detection algorithm refined for higher performance and accuracy on large boards ([check-shorts #7](https://github.com/tscircuit/check-shorts/pull/7))
