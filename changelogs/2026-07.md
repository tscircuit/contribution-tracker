# Changelog 2026-07

- Schematic viewer now highlights the hovered net while dimming all others for easier tracing ([tscircuit/schematic-viewer#237](https://github.com/tscircuit/schematic-viewer/pull/237))
- Default autorouting upgraded to the faster, more-reliable Pipeline 7 ([tscircuit/core#2618](https://github.com/tscircuit/core/pull/2618))
- Autorouter improvements:
  - Correct BGA grid alignment around obstacles ([tscircuit/tscircuit-autorouter#1545](https://github.com/tscircuit/tscircuit-autorouter/pull/1545))
  - Better same-net via merging to reduce excess vias ([#1537](https://github.com/tscircuit/tscircuit-autorouter/pull/1537))
- New differential-pair length-matching solver added ([tscircuit/differential-pairs-solver#1](https://github.com/tscircuit/differential-pairs-solver/pull/1))
- CLI gains `tsci check-shorts` with bitmap-based PCB/Gerber short detection & debug artifacts ([tscircuit/cli#3587](https://github.com/tscircuit/cli/pull/3587), [check-shorts#7](https://github.com/tscircuit/check-shorts/pull/7))
- Gerber exporter upgrades:
  - Preserve original silkscreen text case ([circuit-json-to-gerber#120](https://github.com/tscircuit/circuit-json-to-gerber/pull/120))
  - Render silkscreen graphics as filled regions ([#122](https://github.com/tscircuit/circuit-json-to-gerber/pull/122))
  - Prevent overlapping PCB cutouts ([#123](https://github.com/tscircuit/circuit-json-to-gerber/pull/123))
- Legacy KiCad 5 standalone footprints are now supported in the importer ([kicad-to-circuit-json#160](https://github.com/tscircuit/kicad-to-circuit-json/pull/160))
- MOSFET symbols gain proper orientation variants and configurable port sides, improving schematic clarity ([core#2652](https://github.com/tscircuit/core/pull/2652), [schematic-symbols#437](https://github.com/tscircuit/schematic-symbols/pull/437))
- Crystal nets now warn when routed traces exceed their maximum length ([core#2643](https://github.com/tscircuit/core/pull/2643))
- PCB bounds calculation includes holes, notes, and rotated cut-outs for more accurate exports ([circuit-to-svg#607](https://github.com/tscircuit/circuit-to-svg/pull/607))
