# Changelog 2026-07

Key highlights for 2026-07
- Schematic-viewer #237 – Nets now glow on hover while unrelated wiring/chips fade for easier tracing.
- Core / routing
  – Default switched to Autorouter Pipeline 7 (#2618) with option to ignore existing routes (#2632).
  – Port-facing-direction data fed into trace solver (#2614, #2623, #645).
  – Custom symbols & footprint port-hints supported (#2617, #2629).
  – Fixes for pin-label order, missing pin_number, capacitor handling, placement failures (#2615, #2616, #2633, #2634).
- Autorouter engine
  – Better same-net via merging, BGA grid alignment, endpoint selection (#1537, #1545, #1543).
  – ConnMap-aware Global DRC & topology merge, differential-pair support, via-count surfacing (#1553, #1592, #1583, #1556).
  – Debug/benchmark enhancements, new datasets & snapshots (#1558, #1570, #1580).
- Schematic-trace-solver
  – Regressions fixed and “ratsnest-free” visualizations added (#648, #646).
  – Power/ground label snapping & multi-corridor escapes (#654, #649).
- PCB/Export
  – Accurate PCB bounds counting (#605, #607).
  – Gerber: preserve text case, render graphics, subtract overlapping cut-outs (#120, #122, #123).
  – KiCad: proper cut-out subtraction, mirrored bottom-layer text, legacy footprint support (#365, #373, kicad-to-cj #160).
- Matchpack – New DecouplingCapRowSolver & collision fixes (#153, #155).
- Check-shorts – New package with faster bitmap short detection; CLI “tsci check-shorts” command added (#7-8, cli #3587).
- CLI/Dev-tools – Autorouter debug images & messages, externalized check-shorts (#3612, #3606, #3616).
- IDE/UX – File-type icons, hover fixes, Git/Vite import resolutions in Monaco editor (#17, #24, #21, #28).
- System-block-designer – Built-in AI assistant (#55).
- EEcircuit-engine #7 – Graceful handling of ngspice failures.
- Docs – QR-code silkscreen & custom autorouter-hint guides added (#773, #790).
- Misc – “Sim-only” snippet URLs (#11), MOSFET/switch symbol alignment fixes (#436), new PCB dimension accessors in KiCadTS (#57).
