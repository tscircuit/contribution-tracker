# Changelog 2026-07

Key changes delivered in July 2026

- Schematic-viewer: highlight hovered net by fading unrelated nets & chips (#237)

- Core / Autorouting
  – Pipeline 7 becomes the default autorouter with new global DRC + connMap awareness (#2618, #1553)
  – Option to ignore existing PCB routing when autorouting (#2632)
  – Pass schematic-port facingDirection to both trace solver & autorouter (#2614, schematic-trace-solver #645)
  – Support custom schematic symbols and LED-footprint port overrides (#2629, #2617)
  – Fixes: capacitor/series-res collisions, placement-fails-routing, up/down schFacingDirection, unique custom-port pin_number (#153, #2634, #2623, #2616)

- Autorouter engine (tscircuit-autorouter)
  – Topology merge & same-net via merging improvements (#1532, #1537, #1552)
  – BGA grid alignment, stitch-endpoint snapping, invalid‐overlap & cut-out fixes (#1545, #1536, #1575, #365)
  – New debug surfaces: via count, solver SVG frames, benchmark HTML viewer (#1556, #1565, #1570)
  – Differential-pair support introduced (#1583) and multiple dataset updates (SRJ18, SRJ23, dataset21)

- Schematic trace solver
  – Ratsnest-free visual snapshots (#646) and new corridor escaping for shared net-label blocks (#649)
  – Power/GND net-label auto-snap (#654) and routing regression fixes (#648, #638)

- Circuit-to-SVG
  – Comprehensive PCB bounds now include all element types (#605, #607)
  – Schematic net-label elements get data attributes; hover CSS removed from base SVG (#602)

- Converters & fabrication outputs
  – Gerber: preserve silkscreen text case, render graphic regions, subtract overlapping cut-outs (#120, #122, #123)
  – KiCad: subtract edge-cut openings, mirror bottom-layer silkscreen, preserve text size/thickness, connectivity fixes (#365, #373, #375, #163)
  – BOM CSV: test points automatically skipped (#16)
  – KiCad-to-CJ: legacy footprint & RP2350 support (#160, #162)

- CLI & tooling
  – New  tsci check-shorts  command with bitmap short detection (#3587) plus routing-artifact image output & autorouter debug logs (#3612, #3606, #3610)
  – @tscircuit/check-shorts published with performance/accuracy boosts (#7–#14)

- Libraries & symbols
  – Switch & MOSFET schematic-terminal alignment fixed (#436)
  – Matchpack adds DecouplingCapRowSolver; collision fixes (#155, #153)
  – Props library now accepts element arrays and MOSFET connections (#718, #720)

- Simulation & SPICE
  – EEcircuit-engine: handle ngspice failure rejection correctly (#7); ngspice-spice-engine bumped to 1.7.6 (#24)

- UI / Editor / Docs
  – Monaco-code-editor gains file-type icons, path-normalization fixes, export clean-ups (#17, #28, #23)
  – tscircuit.com: resolves Vercel build regression, native PNG 3-D previews, symbol & package updates (#3850, #3882)
  – Docs: new QR-code silkscreen how-to & custom autorouter hint pages (#773, #790)

- Miscellaneous
  – Check-shorts now uses shared bounds utilities for large-board performance (#8)
  – System-block-designer adds AI chat assistant (#55)
  – SVG service exposes standalone simulation graph URLs (#1763)
