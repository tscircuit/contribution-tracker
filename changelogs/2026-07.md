# Changelog 2026-07

Key highlights for 2026-07

- Schematic & Viewer
  – Hovered-net highlighting with auto-fade for unrelated items (schematic-viewer #237)
  – Switch/MOSFET terminal alignment fixed for cleaner routing (#436)

- Core & Routing
  – Pipeline 7 set as new default with topology merge, BGA grid fix, same-net via merging, differential-pair groundwork, and detailed via-count/debug stats (autorouter #1532, #1545, #1537, #1556, #1583, #1592)
  – Core now passes pin‐facing info to solvers, supports up/down faces, custom footprint port hints, and can ignore existing PCB routes when re-autorouting (#2614, #2623, #2617, #2632)
  – Schematic-trace-solver gains rats-nest-free snapshots, corridor escape logic, power/ground label snapping, and multiple short-routing fixes (#646–#654)
  – Decoupling-row solver added and collision fixes in matchpack (#155, #153)

- Output Generators
  – circuit-to-svg: net-label data-tags, complete PCB bounds (incl. holes/cutouts), unused CSS removed (#602, #605, #607)
  – Gerber: preserves silkscreen text case, outputs silkscreen graphics, resolves overlapping cut-outs (#120, #122, #123)
  – KiCad: proper cut-out subtraction, mirrored bottom-layer text, explicit text thickness (#365, #373, #375)
  – STEP & misc: rotated-pill pad support (#59)

- CLI & Tooling
  – New “tsci check-shorts” command with high-performance bitmap short detection; externalized for downstream use (#3587, #3616)
  – Autorouter debug images & verbose messages now emitted by CLI (#3612, #3606)
  – check-shorts package published with performance/accuracy upgrades (#7, #9, #14)

- Miscellaneous
  – Hover-net labeling, improved schematic symbol support for MOSFET connections (#720) and element-array props (#718)
  – EEcircuit-engine no longer rejects on ngspice failures (#7)
  – Monaco editor gains file-type icons, better path normalization, and visible hover states (#17, #28, #24)

Version bumps & routine dependency updates omitted for brevity.
