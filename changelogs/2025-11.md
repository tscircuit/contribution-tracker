# Changelog 2025-11

2025-11 Key Changes
- Copper Pour Overhaul  – New @tscircuit/copper-pour-solver library with clearance, boardEdgeMargin & via-cutout support (core#1621/#1630, solver#2-26)
- Rectangular Hole Support  – End-to-end handling in circuit-json, props, pcb-viewer, core & circuit-to-svg/gerber (cjson#341/#339, props#491, pcb-viewer#472, core#1635, svg#419)
- PCB Viewer Upgrades  – bottom-layer rectangular pads, corner-radius rectangles, diagonal/origin dimension labels, rotated-snap fix & clearer text offset (pcb-viewer#468/#460/#477/#476/#480/#470)
- Release-Tracker Integration  – Added to schematic-viewer, pcb-viewer, 3d-viewer, runframe & cli for automated version posts (#145, #465, 3d-viewer#553, runframe#1700, cli#883)
- Copper Pour Props  – boardEdgeMargin, cutoutMargin & clearance exposed in props and core (props#488-#490, core#1625)
- PCB Board Flexibility  – width/height now optional for outline boards (cjson#339, pcb-viewer#461, 3d-viewer#554)
- Connectivity Keys  – Via subcircuit_connectivity_map_key rename & support (cjson#342-#343, solver#26)
- Footprinter Enhancements  – TQFP support plus QFN/TQFP pad-size fixes & new snapshot tests (#411, #414-#415)
- 3D Viewer Quality  – faux-board mode, touch-screen menu, session camera restore, silkscreen circle & correct board angle (3d-viewer#564/#562/#557/#555/#551)
- CLI Improvements  – `tsci build --transpile`, dual ESM/CJS JSON import, path handling & glob support (cli#841/#897/#894/#844/#877)
- Simulation & Error Handling  – `simulation_unknown_experiment_error` plus cleaner RunFrame logs (core#1632, core#1642, runframe#1727)
- Docs & Examples  – SPICE simulation guide, copper-pour docs, infinite 3D grid screenshots (docs#317/#316/#321)
- New 3D Models  – TO-220, MELF family, SOD882, MSOP/MS012 and more in jscad-electronics (#195-#211)
- Autorouter  – Off-board obstacle connectivity metadata added (autorouter#302)
- Misc  – Alphabet baseline fix, JLCSearch resistor-array route, kicad(ts) polygon pad support, etc.
