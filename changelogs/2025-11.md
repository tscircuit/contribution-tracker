# Changelog 2025-11

November 2025 – Highlights

Platform-wide
- Added release-tracker hooks to Schematic-Viewer (#145), PCB-Viewer (#465), 3D-Viewer (#553) & RunFrame (#1700) for automatic change-log aggregation.
- CLI: new “--transpile” & ESM/CJS JSON import support (#894, #897) + numerous path/build fixes.

PCB / Board features
- Rectangular plated holes end-to-end support – JSON schema (#341), Props (#491), PCB-Viewer render (#472) & Core processing (#1635).
- Copper-pour overhaul – new boardEdgeMargin, clearance & via-cut-out logic (Core #1621 #1625 #1630, Copper-Pour-Solver series #6–27).
- Optional width/height & polygon shape for PCBBoard (circuit-json #339 #340; Core render assert #1622).
- Dimension tool UX: diagonal labels (#477), origin snapping (#476), rotated snap-points fix (#480), text offset fix (#470).
- Bottom-layer support for rectangular pads (#468) & corner-radius rectangles (#460).

3D & Visualization
- 3D-Viewer: faux-board rendering (#564), touch-screen context menu (#562), session-based camera restore (#557), silkscreen circle support (#555) & load-angle fix (#551).
- Schematic-Viewer: control tooltips (#146).
- SVG viewer: optional infinite grid (#572).

Footprints / Components
- Footprinter: TQFP defaults (#411), MELF/Mini/MicroMELF + SOD123 tests (#406 #407), QFN pad-width & global dimension fixes (#414 #415).
- jscad-electronics: new TO-220 (#211), MSOP/MS012, QFP/LQFP pad length corrections, MELF family, SOD882, SOT723 colour & snapshot updates (#195-#209).

Automation / Routing
- Autorouter: off-board obstacle metadata (#302).
- schematic-trace-solver refactor for label merging (#94).

Simulation / Eval
- Core: simulation_unknown_experiment_error result type (#1632).
- Eval: partsEngineDisabled flag (#1483).

Docs & Tooling
- New SPICE simulation docs (#317) & scripting guide for circuit sizing (#314).
- Docs & RunFrame UI polish – error panel UX (#1672 #1727) & infinite 3D grid screenshots (#321).

Misc
- Circuit-to-SVG & Gerber pipelines now honour PcbHoleCircle/Rect & solder-mask coverage (#419, #62).
- kicad-to-json/ts: polygon smt-pads & graphics primitives (#18-23).
- Contribution-tracker: sponsorship tier increase (#255) and user additions.

(Countless version bumps & dependency updates omitted.)
