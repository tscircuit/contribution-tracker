# Changelog 2025-11

- **PCB Viewer**
  - Add bottom-layer rendering for plated-hole rectangular pads ([[tscircuit/pcb-viewer #468](https://github.com/tscircuit/pcb-viewer/pull/468)](https://github.com/tscircuit/pcb-viewer/pull/468))
  - Support `corner_radius` on rectangle primitives ([#460](https://github.com/tscircuit/pcb-viewer/pull/460))
  - Allow PCB boards without explicit `width/height` ([#461](https://github.com/tscircuit/pcb-viewer/pull/461))

- **Circuit-JSON**
  - `width/height` now optional for outline-defined boards ([#339](https://github.com/tscircuit/circuit-json/pull/339))
  - New optional `shape` property on `PCBBoard` ([#340](https://github.com/tscircuit/circuit-json/pull/340))
  - Added `PcbHoleCircle` & `PcbHoleRect` elements ([#341](https://github.com/tscircuit/circuit-json/pull/341))
  - Added/renamed via connectivity keys for subcircuits ([#342](https://github.com/tscircuit/circuit-json/pull/342), [#343](https://github.com/tscircuit/circuit-json/pull/343))

- **Props**
  - Introduce `RectHoleProps` for rectangular holes ([[tscircuit/props #491](https://github.com/tscircuit/props/pull/491)](https://github.com/tscircuit/props/pull/491))
  - Add `minTraceWidth` to groups ([#489](https://github.com/tscircuit/props/pull/489))
  - Add `cutoutMargin` & `boardEdgeMargin` to copper pours ([#490](https://github.com/tscircuit/props/pull/490), [#488](https://github.com/tscircuit/props/pull/488))

- **Footprinter**

- **3D Viewer**
  - Render `pcb_silkscreen_circle` elements ([[tscircuit/3d-viewer #555](https://github.com/tscircuit/3d-viewer/pull/555)](https://github.com/tscircuit/3d-viewer/pull/555))

  - Fix boards loading at wrong angle ([#551](https://github.com/tscircuit/3d-viewer/pull/551))

- **Core**
  - Copper-pour overhaul: new solver, `boardEdgeMargin`/`clearance`, via cut-outs, polygon-board support, net creation & solder-mask coverage ([#1621](https://github.com/tscircuit/core/pull/1621), [#1625](https://github.com/tscircuit/core/pull/1625), [#1630](https://github.com/tscircuit/core/pull/1630), [#1634](https://github.com/tscircuit/core/pull/1634), [#1636](https://github.com/tscircuit/core/pull/1636), [#1627](https://github.com/tscircuit/core/pull/1627))
  - Support rectangular PCB holes ([#1635](https://github.com/tscircuit/core/pull/1635))
  - Added `simulation_unknown_experiment_error` for failed sims ([#1632](https://github.com/tscircuit/core/pull/1632))
