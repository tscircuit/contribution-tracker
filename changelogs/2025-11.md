# Changelog 2025-11

- Copper-pour improvements: clearance & boardEdgeMargin support, via cut-outs, solder-mask coverage and polygon-board fixes ([core #1625](https://github.com/tscircuit/core/pull/1625), [core #1630](https://github.com/tscircuit/core/pull/1630), [circuit-json-to-gerber #62](https://github.com/tscircuit/circuit-json-to-gerber/pull/62))
- New hole types: rectangular and polygon-pad plated holes now fully handled across schema and viewers ([circuit-json #341](https://github.com/tscircuit/circuit-json/pull/341), [pcb-viewer #472](https://github.com/tscircuit/pcb-viewer/pull/472), [circuit-to-svg #420](https://github.com/tscircuit/circuit-to-svg/pull/420), [core #1635](https://github.com/tscircuit/core/pull/1635))
- Board outline flexibility: width/height now optional & shape field added for outlined boards ([circuit-json #339](https://github.com/tscircuit/circuit-json/pull/339), [circuit-json #340](https://github.com/tscircuit/circuit-json/pull/340))
- PCB panel enhancements: new center property and viewer support ([circuit-json #344](https://github.com/tscircuit/circuit-json/pull/344), [pcb-viewer #473](https://github.com/tscircuit/pcb-viewer/pull/473), [circuit-to-svg #421](https://github.com/tscircuit/circuit-to-svg/pull/421))
- Dimensioning UX: origin-snap for dimension tool and clearer perpendicular text placement ([pcb-viewer #476](https://github.com/tscircuit/pcb-viewer/pull/476), [pcb-viewer #470](https://github.com/tscircuit/pcb-viewer/pull/470))
- Rectangle primitives gain corner-radius support ([pcb-viewer #460](https://github.com/tscircuit/pcb-viewer/pull/460))
- Bottom-layer support for plated-hole rectangular pads ([pcb-viewer #468](https://github.com/tscircuit/pcb-viewer/pull/468))
- 3D viewer quality-of-life: touch-screen context menu, faux-board rendering & persistent/orthogonal cameras ([3d-viewer #562](https://github.com/tscircuit/3d-viewer/pull/562), [3d-viewer #564](https://github.com/tscircuit/3d-viewer/pull/564), [3d-viewer #569](https://github.com/tscircuit/3d-viewer/pull/569))
- Voltage probe labels: new name property plus improved label alignment in schematics & SVG export ([circuit-json #347](https://github.com/tscircuit/circuit-json/pull/347), [circuit-to-svg #422](https://github.com/tscircuit/circuit-to-svg/pull/422), [circuit-to-svg #423](https://github.com/tscircuit/circuit-to-svg/pull/423))
- CLI gains `tsci install`, JSON import for both ESM/CJS, and transpilation support ([cli #923](https://github.com/tscircuit/cli/pull/923), [cli #897](https://github.com/tscircuit/cli/pull/897), [cli #894](https://github.com/tscircuit/cli/pull/894))
- Eval/props: introduce `partsEngineDisabled` and `nodeModulesResolver` options for advanced builds ([eval #1483](https://github.com/tscircuit/eval/pull/1483), [props #495](https://github.com/tscircuit/props/pull/495))
- Direct KiCad PCB import now supported in Eval ([eval #1495](https://github.com/tscircuit/eval/pull/1495))
