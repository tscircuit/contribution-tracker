# Changelog 2025-11

Key updates delivered in November 2025

General platform
- Release-tracker auto-posting rolled out across major apps – schematic-viewer #145, pcb-viewer #465, 3d-viewer #553, runframe #1700, cli #883.
- CLI gained first-class transpilation & JSON-import support (cli #841, #894, #897) and improved path handling (cli #877).

PCB & mechanical features
- Copper-pour pipeline shipped: new solver package (copper-pour-solver #2-#26) and Core integration with clearance, boardEdgeMargin & cutoutMargin (core #1621, #1625, #1630, #1636-#1637).
- Rectangular hole support end-to-end – schema (circuit-json #341), core rendering (core #1635), svg/gerber output (circuit-to-svg #419), viewers (pcb-viewer #472, 3d-viewer #555).
- Optional PCBBoard width/height & new shape prop (circuit-json #339-#340) plus viewer fallback fix (pcb-viewer #461, 3d-viewer #554).
- Bottom-layer plated-hole rectangular pads supported (pcb-viewer #468).
- Dimension tool polish – diagonal labels (pcb-viewer #477), rotated-snap fix (pcb-viewer #480), origin snap for dimensions (pcb-viewer #476).

Visualization
- 3D-viewer upgrades: faux-board mode (3d-viewer #564), session camera persistence (3d-viewer #557), touch-screen context menus (3d-viewer #562), better default angles (3d-viewer #551-#552).
- Schematic-viewer usability: added tooltips to controls (schematic-viewer #146).

Data / props ecosystem
- New props: boardEdgeMargin & clearance for copper pours (props #488-#490), minTraceWidth in groups (#489), optional children for footprints (#493).
- pcb_via connectivity overhaul, incl. subcircuit key (circuit-json #342-#343).

Manufacturing & libraries
- Footprinter: TQFP family, MELF/MicroMELF/MiniMELF, pad-dimension fixes (footprinter #406-#417).
- jscad-electronics: TO-220, SOD882, MSOP/MS012, MELF series, color / dimension fixes (#195-#211).
- circuit-json-to-gerber: polygon smtpad & copper-pour solder-mask support (#62-#64).

Tooling & docs
- Runframe UI improvements – collapsible error lists, natural sort & layout fixes (runframe #1727, #1672-#1673).
- Docs: new SPICE simulation guide, infinite 3D grid screenshots, copper-pour docs updates (docs #314-#317, #321).

Infrastructure
- Bun 1.3.1 pinning across CI pipelines to stabilise tests (multiple repos).
