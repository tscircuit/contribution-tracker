# Changelog 2025-11

- Release-tracker hooks added across key UIs — schematic-viewer [#145](https://github.com/tscircuit/schematic-viewer/pull/145), pcb-viewer [#465](https://github.com/tscircuit/pcb-viewer/pull/465), 3d-viewer [#553](https://github.com/tscircuit/3d-viewer/pull/553) and runframe [#1700](https://github.com/tscircuit/runframe/pull/1700)
- PCB-viewer upgrades: rectangle `corner_radius` support [#460](https://github.com/tscircuit/pcb-viewer/pull/460) and boards can now omit explicit width/height [#461](https://github.com/tscircuit/pcb-viewer/pull/461)
- 3D-viewer improvements: silkscreen circles [#555](https://github.com/tscircuit/3d-viewer/pull/555), corrected board angle [#551](https://github.com/tscircuit/3d-viewer/pull/551) and smarter default camera [#552](https://github.com/tscircuit/3d-viewer/pull/552)
- Copper-pour revamp
  - Core adds `boardEdgeMargin`, `clearance` [#1625](https://github.com/tscircuit/core/pull/1625) and `cutoutMargin` / via cutouts [#1630](https://github.com/tscircuit/core/pull/1630)
  - New solver pipeline in copper-pour-solver [#2](https://github.com/tscircuit/copper-pour-solver/pull/2) with board-edge margin [#6](https://github.com/tscircuit/copper-pour-solver/pull/6), via cutouts [#18](https://github.com/tscircuit/copper-pour-solver/pull/18) and cutout clearance [#20](https://github.com/tscircuit/copper-pour-solver/pull/20)
  - Gerber exporter gains solder-mask flag [#62](https://github.com/tscircuit/circuit-json-to-gerber/pull/62) and polygon smt-pad support [#63](https://github.com/tscircuit/circuit-json-to-gerber/pull/63)
- New PCB schema options: optional board width/height [circuit-json #339](https://github.com/tscircuit/circuit-json/pull/339) and `shape` field [#340](https://github.com/tscircuit/circuit-json/pull/340)
- props package: `RectHoleProps` [#491](https://github.com/tscircuit/props/pull/491), `boardEdgeMargin` [#488](https://github.com/tscircuit/props/pull/488), `cutoutMargin` [#490](https://github.com/tscircuit/props/pull/490) and group-level `minTraceWidth` [#489](https://github.com/tscircuit/props/pull/489)
- CLI additions: transpilation flag `tsci build --transpile` [cli #841](https://github.com/tscircuit/cli/pull/841) and glob support in `tsci snapshot` [#844](https://github.com/tscircuit/cli/pull/844)
- KiCad ↔︎ Circuit-JSON parity
  - kicad-to-circuit-json: polygon smt-pads [#23](https://github.com/tscircuit/kicad-to-circuit-json/pull/23), rect pads fix [#22](https://github.com/tscircuit/kicad-to-circuit-json/pull/22), ViaGridBoard [#20](https://github.com/tscircuit/kicad-to-circuit-json/pull/20)
  - circuit-json-to-kicad: polygon smtpads [#29](https://github.com/tscircuit/circuit-json-to-kicad/pull/29)
  - kicadts adds GrCircle/Rect/Poly & graphic groups [#18](https://github.com/tscircuit/kicadts/pull/18)–[#21](https://github.com/tscircuit/kicadts/pull/21)
- New footprints & 3D models
  - Footprinter: MELF/MicroMELF/MiniMELF footprints & tests [#406](https://github.com/tscircuit/footprinter/pull/406), SOD123 parity test [#407](https://github.com/tscircuit/footprinter/pull/407)
  - jscad-electronics: MELF series [#195](https://github.com/tscircuit/jscad-electronics/pull/195)–[#197](https://github.com/tscircuit/jscad-electronics/pull/197), SOD882 [#202](https://github.com/tscircuit/jscad-electronics/pull/202), MS012 [#199](https://github.com/tscircuit/jscad-electronics/pull/199)
- Core quality-of-life: copper-pour nets created from `connectsTo` selectors [#1627](https://github.com/tscircuit/core/pull/1627), silkscreen font-size override visual tests [#1620](https://github.com/tscircuit/core/pull/1620), footprints now receive resolved PCB style [#1617](https://github.com/tscircuit/core/pull/1617)
