# Changelog 2025-11

- Rectangular plated holes now fully supported across the stack ([circuit-json#341](https://github.com/tscircuit/circuit-json/pull/341), [core#1635](https://github.com/tscircuit/core/pull/1635), [pcb-viewer#468](https://github.com/tscircuit/pcb-viewer/pull/468))
- Copper-pour upgrades: board-edge & cut-out margins, via cut-outs, clearance control, and solder-mask reporting ([core#1625](https://github.com/tscircuit/core/pull/1625), [core#1630](https://github.com/tscircuit/core/pull/1630), [core#1634](https://github.com/tscircuit/core/pull/1634))
- PCB Viewer improvements:
  • Rounded-corner rectangles ([pcb-viewer#460](https://github.com/tscircuit/pcb-viewer/pull/460))
  • Dimension text auto-offset ([pcb-viewer#470](https://github.com/tscircuit/pcb-viewer/pull/470))
  • Board width/height now optional ([pcb-viewer#461](https://github.com/tscircuit/pcb-viewer/pull/461))
- 3D Viewer enhancements:
  • Touch-screen context menu ([3d-viewer#562](https://github.com/tscircuit/3d-viewer/pull/562))
  • Session-based camera persistence ([3d-viewer#557](https://github.com/tscircuit/3d-viewer/pull/557))
  • Correct board orientation on load ([3d-viewer#551](https://github.com/tscircuit/3d-viewer/pull/551))
  • Silkscreen circles & faux-board rendering ([3d-viewer#555](https://github.com/tscircuit/3d-viewer/pull/555), [3d-viewer#564](https://github.com/tscircuit/3d-viewer/pull/564))
- Schematic Viewer: tooltips added to UI controls ([schematic-viewer#146](https://github.com/tscircuit/schematic-viewer/pull/146))
- CLI updates:
  • New `tsci install` command ([cli#923](https://github.com/tscircuit/cli/pull/923))
  • ESM/CJS JSON transpilation support ([cli#897](https://github.com/tscircuit/cli/pull/897), [cli#894](https://github.com/tscircuit/cli/pull/894))
- Eval: import KiCad `.kicad_pcb` files and optionally disable Parts Engine ([eval#1495](https://github.com/tscircuit/eval/pull/1495), [eval#1483](https://github.com/tscircuit/eval/pull/1483))
- Circuit-JSON schema tweaks: optional board size, new `shape` & `center` fields, VIA connectivity key rename & subcircuit key ([circuit-json#339](https://github.com/tscircuit/circuit-json/pull/339), [circuit-json#340](https://github.com/tscircuit/circuit-json/pull/340), [circuit-json#344](https://github.com/tscircuit/circuit-json/pull/344), [circuit-json#343](https://github.com/tscircuit/circuit-json/pull/343), [circuit-json#342](https://github.com/tscircuit/circuit-json/pull/342))
- New footprints & 3D models: TQFP, TO-220, SOT-363, MELF/MicroMELF/MiniMELF, SOD882, MSOP and more ([footprinter#411](https://github.com/tscircuit/footprinter/pull/411), [jscad-electronics#211](https://github.com/tscircuit/jscad-electronics/pull/211), [jscad-electronics#219](https://github.com/tscircuit/jscad-electronics/pull/219), [jscad-electronics#195](https://github.com/tscircuit/jscad-electronics/pull/195), [jscad-electronics#197](https://github.com/tscircuit/jscad-electronics/pull/197), [jscad-electronics#196](https://github.com/tscircuit/jscad-electronics/pull/196), [jscad-electronics#202](https://github.com/tscircuit/jscad-electronics/pull/202))
