# Changelog 2025-11

- PCB Viewer: bottom-layer rendering for plated-hole rectangular pads — [#468](https://github.com/tscircuit/pcb-viewer/pull/468)
- PCB Viewer: rectangle primitives now accept a `corner_radius` prop — [#460](https://github.com/tscircuit/pcb-viewer/pull/460)
- PCB Viewer: dimension text is now correctly offset perpendicular to its line — [#470](https://github.com/tscircuit/pcb-viewer/pull/470)
- Circuit-JSON: new `PcbHoleCircle` & `PcbHoleRect` element types — [#341](https://github.com/tscircuit/circuit-json/pull/341)
- Circuit-JSON: boards can omit width/height & define a `shape`, enabling pure-outline boards — [#339](https://github.com/tscircuit/circuit-json/pull/339), [#340](https://github.com/tscircuit/circuit-json/pull/340)
- Circuit-JSON: vias can reference subcircuits via `subcircuit_connectivity_key` — [#342](https://github.com/tscircuit/circuit-json/pull/342)
- Core: copper-pour overhaul — boardEdgeMargin/clearance props, via cut-outs, polygon-board fixes, solder-mask coverage — [#1625](https://github.com/tscircuit/core/pull/1625), [#1630](https://github.com/tscircuit/core/pull/1630), [#1636](https://github.com/tscircuit/core/pull/1636), [#1634](https://github.com/tscircuit/core/pull/1634)
- Core: support for rectangular board holes — [#1635](https://github.com/tscircuit/core/pull/1635)
- Core: clearer simulation errors via `simulation_unknown_experiment_error` — [#1632](https://github.com/tscircuit/core/pull/1632)
- 3D Viewer: touch-screen context menu — [#562](https://github.com/tscircuit/3d-viewer/pull/562)
- 3D Viewer: session-based camera persistence & early restore — [#557](https://github.com/tscircuit/3d-viewer/pull/557)
- 3D Viewer: faux-board rendering and silkscreen-circle support — [#564](https://github.com/tscircuit/3d-viewer/pull/564), [#555](https://github.com/tscircuit/3d-viewer/pull/555)
- circuit-to-svg: rendering for rectangular/circular holes + `showAnchorOffsets` option — [#419](https://github.com/tscircuit/circuit-to-svg/pull/419), [#416](https://github.com/tscircuit/circuit-to-svg/pull/416)
- circuit-json-to-gerber: copper pours honour `covered_with_solder_mask`; export polygon SMT pads — [#62](https://github.com/tscircuit/circuit-json-to-gerber/pull/62), [#63](https://github.com/tscircuit/circuit-json-to-gerber/pull/63)
- Footprinter & CAD libs: added TQFP, TO-220, MELF family, MS012, SOD882 & Micro/Minimelf models; fixed QFN/TQFP pad sizing — [footprinter #411](https://github.com/tscircuit/footprinter/pull/411), [jscad-electronics #211](https://github.com/tscircuit/jscad-electronics/pull/211), [#195](https://github.com/tscircuit/jscad-electronics/pull/195) and related PRs
