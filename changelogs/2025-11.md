# Changelog 2025-11

- Rectangle component now supports rounded corners ([pcb-viewer #460](https://github.com/tscircuit/pcb-viewer/pull/460))
- PCB boards can be defined without width/height in viewer & JSON schema ([pcb-viewer #461](https://github.com/tscircuit/pcb-viewer/pull/461), [circuit-json #339](https://github.com/tscircuit/circuit-json/pull/339))
- Boards load at the correct angle in the 3-D viewer ([3d-viewer #551](https://github.com/tscircuit/3d-viewer/pull/551))
- Copper-pour improvements:
  - New solver pipeline ([copper-pour-solver #2](https://github.com/tscircuit/copper-pour-solver/pull/2), core [#1621](https://github.com/tscircuit/core/pull/1621))
  - boardEdgeMargin prop supported end-to-end ([props #488](https://github.com/tscircuit/props/pull/488), [copper-pour-solver #6](https://github.com/tscircuit/copper-pour-solver/pull/6))
  - `covered_with_solder_mask` exported to Gerber ([circuit-json-to-gerber #62](https://github.com/tscircuit/circuit-json-to-gerber/pull/62))
- KiCad compatibility extended:
  - New graphic element types (GrCircle, GrRect, GrPoly, Group, graphicsPoly/Rect) ([kicadts #18](https://github.com/tscircuit/kicadts/pull/18), #19, #20, #21)
  - Export: polygon SMT pads now supported ([circuit-json-to-kicad #29](https://github.com/tscircuit/circuit-json-to-kicad/pull/29))
  - Import: gr_poly parsing, rect SMT pads, ViaGridBoard added ([kicad-to-circuit-json #21](https://github.com/tscircuit/kicad-to-circuit-json/pull/21), #22, #20)
- Footprint loaders now honor resolved PCB styles for KiCad footprints ([core #1617](https://github.com/tscircuit/core/pull/1617))
- New MS012 component & example in JSCAD-electronics ([jscad-electronics #199](https://github.com/tscircuit/jscad-electronics/pull/199))
- Runframe UX tweaks: natural filename sort & console-style expandable error list ([runframe #1673](https://github.com/tscircuit/runframe/pull/1673), [#1672](https://github.com/tscircuit/runframe/pull/1672))
- CLI improvements:
  - `tsci build --transpile` flag added ([cli #841](https://github.com/tscircuit/cli/pull/841))
  - `tsci snapshot` now accepts glob patterns ([cli #844](https://github.com/tscircuit/cli/pull/844))
- Fixed proxy URL mis-routing in tscircuit.com import dialog ([tscircuit.com #1952](https://github.com/tscircuit/tscircuit.com/pull/1952))
