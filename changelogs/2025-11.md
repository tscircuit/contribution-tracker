# Changelog 2025-11

- Release-tracker integrated into all major viewers and RunFrame ([schematic-viewer#145](https://github.com/tscircuit/schematic-viewer/pull/145), [pcb-viewer#465](https://github.com/tscircuit/pcb-viewer/pull/465), [3d-viewer#553](https://github.com/tscircuit/3d-viewer/pull/553), [runframe#1700](https://github.com/tscircuit/runframe/pull/1700))
- Copper pour overhaul: new boardEdgeMargin, clearance & cutoutMargin props, via/cutout handling, nets from connectsTo, solder-mask coverage & solver pipeline ([props#488](https://github.com/tscircuit/props/pull/488), [props#490](https://github.com/tscircuit/props/pull/490), [core#1630](https://github.com/tscircuit/core/pull/1630), [core#1625](https://github.com/tscircuit/core/pull/1625), [core#1627](https://github.com/tscircuit/core/pull/1627), [core#1634](https://github.com/tscircuit/core/pull/1634), [copper-pour-solver#2](https://github.com/tscircuit/copper-pour-solver/pull/2))
- Copper pours now export correct solder-mask data to Gerber ([circuit-json-to-gerber#62](https://github.com/tscircuit/circuit-json-to-gerber/pull/62))
- PCB board geometry is more flexible: width/height optional for outlined boards & new shape attribute ([circuit-json#339](https://github.com/tscircuit/circuit-json/pull/339), [circuit-json#340](https://github.com/tscircuit/circuit-json/pull/340), [pcb-viewer#461](https://github.com/tscircuit/pcb-viewer/pull/461))
- Rectangles component now supports corner_radius ([pcb-viewer#460](https://github.com/tscircuit/pcb-viewer/pull/460))
- Added circular & rectangular hole types across the stack ([circuit-json#341](https://github.com/tscircuit/circuit-json/pull/341), [props#491](https://github.com/tscircuit/props/pull/491), [circuit-to-svg#419](https://github.com/tscircuit/circuit-to-svg/pull/419))
- PCB SMT pads can now be polygons; Gerber exporter updated accordingly ([circuit-json-to-kicad#29](https://github.com/tscircuit/circuit-json-to-kicad/pull/29), [circuit-json-to-gerber#63](https://github.com/tscircuit/circuit-json-to-gerber/pull/63))
- 3D Viewer improvements: silkscreen circle support, correct board orientation & better default camera ([3d-viewer#555](https://github.com/tscircuit/3d-viewer/pull/555), [3d-viewer#551](https://github.com/tscircuit/3d-viewer/pull/551), [3d-viewer#552](https://github.com/tscircuit/3d-viewer/pull/552))
- Footprinter adds TQFP footprints with sensible defaults ([footprinter#411](https://github.com/tscircuit/footprinter/pull/411))
- New simulation_unknown_experiment_error clarifies failed simulation feedback ([core#1632](https://github.com/tscircuit/core/pull/1632))
- CLI enhancements: `tsci build --transpile`, glob support for `tsci snapshot`, and more robust path handling ([cli#841](https://github.com/tscircuit/cli/pull/841), [cli#844](https://github.com/tscircuit/cli/pull/844), [cli#877](https://github.com/tscircuit/cli/pull/877))
- JLCSearch API gains resistor array category and understands C-prefixed part numbers ([jlcsearch#108](https://github.com/tscircuit/jlcsearch/pull/108), [jlcsearch#107](https://github.com/tscircuit/jlcsearch/pull/107))
- tscircuit.com UX updates: better organization search/filter, email privacy for non-authenticated users, tscircuit_handle support, improved meta tags & default package view ([tscircuit.com#1970](https://github.com/tscircuit/tscircuit.com/pull/1970), [tscircuit.com#1969](https://github.com/tscircuit/tscircuit.com/pull/1969), [tscircuit.com#1968](https://github.com/tscircuit/tscircuit.com/pull/1968), [tscircuit.com#1961](https://github.com/tscircuit/tscircuit.com/pull/1961), [tscircuit.com#1965](https://github.com/tscircuit/tscircuit.com/pull/1965))
