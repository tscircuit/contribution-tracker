# Changelog 2025-11

- PCB-viewer
  - Added bottom-layer support for plated rectangular pads [#468](https://github.com/tscircuit/pcb-viewer/pull/468)
  - Rectangle primitives now accept corner_radius [#460](https://github.com/tscircuit/pcb-viewer/pull/460)
  - Dimension text is now correctly offset from the line [#470](https://github.com/tscircuit/pcb-viewer/pull/470)
  - Gracefully handles boards without explicit width/height [#461](https://github.com/tscircuit/pcb-viewer/pull/461)

- Board & hole schema
  - Rectangular & circular board holes fully supported across schema, SVG and Gerber export [circuit-json #341](https://github.com/tscircuit/circuit-json/pull/341), [circuit-to-svg #419](https://github.com/tscircuit/circuit-to-svg/pull/419), [circuit-json-to-gerber #63](https://github.com/tscircuit/circuit-json-to-gerber/pull/63), [core #1635](https://github.com/tscircuit/core/pull/1635)
  - Boards can now omit width/height when an outline is supplied; optional shape property added [circuit-json #339](https://github.com/tscircuit/circuit-json/pull/339) & [#340](https://github.com/tscircuit/circuit-json/pull/340)

- Copper-pour improvements
  - New boardEdgeMargin & cutoutMargin props, via cutouts and clearance for board cut-outs [props #488](https://github.com/tscircuit/props/pull/488), [#490](https://github.com/tscircuit/props/pull/490), [core #1625](https://github.com/tscircuit/core/pull/1625), [#1630](https://github.com/tscircuit/core/pull/1630), [copper-pour-solver #20](https://github.com/tscircuit/copper-pour-solver/pull/20), [#26](https://github.com/tscircuit/copper-pour-solver/pull/26), [circuit-json-to-gerber #62](https://github.com/tscircuit/circuit-json-to-gerber/pull/62)

- 3D-viewer
  - Touch-screen context menu support [#562](https://github.com/tscircuit/3d-viewer/pull/562)
  - Session-based camera persistence & early restore [#557](https://github.com/tscircuit/3d-viewer/pull/557)
  - Added faux-board rendering [#564](https://github.com/tscircuit/3d-viewer/pull/564)
  - Orthogonal camera option & refactor to useCameraController [#569](https://github.com/tscircuit/3d-viewer/pull/569)
  - Better default camera angle and correct board orientation [#552](https://github.com/tscircuit/3d-viewer/pull/552), [#551](https://github.com/tscircuit/3d-viewer/pull/551)
  - Support for silkscreen circles [#555](https://github.com/tscircuit/3d-viewer/pull/555)

- Core engine
  - Sub-circuit JSON now supported for capacitors, inductors and diodes [#1648](https://github.com/tscircuit/core/pull/1648), [#1649](https://github.com/tscircuit/core/pull/1649), [#1651](https://github.com/tscircuit/core/pull/1651)
  - New simulation_unknown_experiment_error status surfaced on failed simulations [#1632](https://github.com/tscircuit/core/pull/1632)
  - Added center data for PCB panels [circuit-json #344](https://github.com/tscircuit/circuit-json/pull/344), [core #1650](https://github.com/tscircuit/core/pull/1650)

- Footprinter
  - TQFP footprint generation [#411](https://github.com/tscircuit/footprinter/pull/411)
  - Corrected QFN & SOT-363 pad dimensions [#415](https://github.com/tscircuit/footprinter/pull/415), [#418](https://github.com/tscircuit/footprinter/pull/418)

- 3D CAD models (jscad-electronics)
  - New TO-220, TO-92 and SOT-363 packages [#211](https://github.com/tscircuit/jscad-electronics/pull/211), [#214](https://github.com/tscircuit/jscad-electronics/pull/214), [#219](https://github.com/tscircuit/jscad-electronics/pull/219)

- Schematic viewer now shows tooltips on controls [#146](https://github.com/tscircuit/schematic-viewer/pull/146)

- Graphics-debug: added arrow graphics element [#80](https://github.com/tscircuit/graphics-debug/pull/80)
