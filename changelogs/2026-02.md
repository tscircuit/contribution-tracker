# Changelog 2026-02

- Courtyard overhaul – full support for pcb_courtyard (rect/circle/outline, top & bottom) added across core, circuit-json, viewers (pcb-viewer, 3d-viewer), SVG/Canvas/KiCad converters.
- Introduced pcbSx shorthand props (props #584, core #1908) with docs & viewer support for easier pin-centric coordinate entry.
- Knock-out silkscreen text rendering supported in SVG & Canvas (svg #506, canvas #159) with core mapping (core #1886).
- Vias now accepted in manual pcbPath definitions (props #578, core #1899) plus trace overlap DRC fixes (core #1916, checks #94).
- PCB-viewer fixes: traces no longer cover plated holes (#646), outline/min-stroke visibility restored (#652), solder-mask trace color tweak (#648).
- 3D-viewer performance/quality: board-size-aware texture caps (#672), STEP model blob-cache (#675), GPU leak plug on cleanup (#673), relative asset resolver (#680).
- New IsolatedCircuit component (core #1911) and autorouting/auto-layout disabled for inflated subcircuits (#1903) improve hierarchical design control.
- Mounted board & screen enhancements: new screen option (footprinter #499/#496) and keep-out sizing util (core #1914).
- CLI quality-of-life: tsci doctor diagnostics cmd (#1893), fatalErrors now exit non-zero (#1879), improved asset path handling (#1916, #1936).
- Graphics-debug gains polygon support & per-polygon stroke widths (#95–97), enabling richer interactive visuals.
- Contribution-tracker cron/start-day tweaks (#303/#301) and status init script fix (#65) bolster infrastructure reliability.
- Docs refreshed: new pcbSx guide, SPICE element docs, KiCad export tutorial cleanup, plus added package descriptions (docs #429–435).
