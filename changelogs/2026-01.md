# Changelog 2026-01

- EasyEDA import now recognises pin labels ending in “#” ([easyeda-converter #341](https://github.com/tscircuit/easyeda-converter/pull/341))
- Major jumper autorouting support added and enabled in the new Pipeline 3, with curvy-trace optimisation and port-point relaxation for cleaner results ([core #1821](https://github.com/tscircuit/core/pull/1821), [autorouter #510](https://github.com/tscircuit/tscircuit-autorouter/pull/510), [autorouter #528](https://github.com/tscircuit/tscircuit-autorouter/pull/528), [autorouter #540](https://github.com/tscircuit/tscircuit-autorouter/pull/540))
- Solder-mask margin can now be set per-side on SMT pads and is honoured across Canvas, SVG and PCB-viewer renderers ([circuit-to-canvas #117](https://github.com/tscircuit/circuit-to-canvas/pull/117), [circuit-to-canvas #119](https://github.com/tscircuit/circuit-to-canvas/pull/119), [circuit-to-svg #481](https://github.com/tscircuit/circuit-to-svg/pull/481), [props #554](https://github.com/tscircuit/props/pull/554))
- Keep-out, copper pour and via rendering upgraded for higher fidelity and correct layer ordering in viewers ([circuit-to-svg #477](https://github.com/tscircuit/circuit-to-svg/pull/477), [pcb-viewer #595](https://github.com/tscircuit/pcb-viewer/pull/595), [pcb-viewer #604](https://github.com/tscircuit/pcb-viewer/pull/604))
- 3D Viewer improvements: STEP-to-GLB fallback, auto-placement of CAD parts, and support for oval / rotated-pill holes ([3d-viewer #638](https://github.com/tscircuit/3d-viewer/pull/638), [3d-viewer #651](https://github.com/tscircuit/3d-viewer/pull/651), [3d-viewer #659](https://github.com/tscircuit/3d-viewer/pull/659))
- CLI quality-of-life: configurable buildCommand / prebuildCommand, `--include-dist`, `--ci` flag, richer push summaries and option to always use the latest cloud package ([cli #1570](https://github.com/tscircuit/cli/pull/1570), [cli #1565](https://github.com/tscircuit/cli/pull/1565), [cli #1577](https://github.com/tscircuit/cli/pull/1577), [cli #1617](https://github.com/tscircuit/cli/pull/1617))
- Kicad export gets new library-conversion API, integrated into the CLI for easier kicad-library generation ([circuit-json-to-kicad #47](https://github.com/tscircuit/circuit-json-to-kicad/pull/47), [cli #1579](https://github.com/tscircuit/cli/pull/1579))
- Footprinter adds convex/concave 0402×2 variants, USB module support and smarter resistor-array naming ([footprinter #471](https://github.com/tscircuit/footprinter/pull/471), [footprinter #480](https://github.com/tscircuit/footprinter/pull/480), [footprinter #481](https://github.com/tscircuit/footprinter/pull/481))
- Core now supports <schematicpath /> primitives and emits warnings on conflicting solder-mask settings ([core #1835](https://github.com/tscircuit/core/pull/1835), [core #1829](https://github.com/tscircuit/core/pull/1829))
- pcbburn.com UX: loading & error states on upload, auto-fit SVG preview, per-cent solder-mask margin control, mobile responsive layout and viewport/selection fixes ([pcbburn.com #17](https://github.com/tscircuit/pcbburn.com/pull/17), [pcbburn.com #25](https://github.com/tscircuit/pcbburn.com/pull/25), [pcbburn.com #28](https://github.com/tscircuit/pcbburn.com/pull/28), [pcbburn.com #29](https://github.com/tscircuit/pcbburn.com/pull/29), [pcbburn.com #30](https://github.com/tscircuit/pcbburn.com/pull/30))
- tscircuit.com adds Builds page, Releases list, revamped release details and manual GitHub resync button ([tscircuit.com #2441](https://github.com/tscircuit/tscircuit.com/pull/2441), [tscircuit.com #2446](https://github.com/tscircuit/tscircuit.com/pull/2446), [tscircuit.com #2471](https://github.com/tscircuit/tscircuit.com/pull/2471), [tscircuit.com #2453](https://github.com/tscircuit/tscircuit.com/pull/2453))
