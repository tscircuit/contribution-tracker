# Changelog 2026-01

- Autorouter: Introduced jumper-aware routing pipelines, straight-line deviation penalties, keep-out solver fixes and optional curvy-trace solver – delivering higher quality, high-density routes (tscircuit-autorouter #506, #510, #515, #520, #528, #529)
- Core / Circuit-JSON: Added jumper connectivity, schematic_path support, new solder-mask properties and autorouterVersion flag; emits warnings for mask conflicts (core #1824, #1826, #1829, #1831, circuit-json #410)
- Hypergraph engine: Gradient-descent optimisation, bounds fixes and serialisation API to drive advanced jumper routing (hypergraph #5, #6, #8, #20-23)
- Rendering stack: circuit-to-canvas/SVG gain solder-mask margins, copper-pour BREP, explicit viewport control, diagonal keep-out patterns; pcb-viewer migrates to new primitives with via & plated-hole hover states (#102-115, pcb-viewer #590-606)
- 3D-viewer: STEP→GLB fallback, correct pill/oval hole handling and full-outline solder-mask clearance (#638, #647, #651-655)
- CLI 0.1.746+: buildCommand/ prebuildCommand, --ci flag, push summaries, KicadLibrary export, clone fixes and “use latest tscircuit on cloud” option (cli #1565-1609, #1617-1631)
- EasyEDA converter: Correctly parse pin labels with “#” suffix, adds tests (#341)
- Kicad / LBRN exporters: New API for kicad-library separation, plated-hole pad merges, version bumps (#47-50, circuit-json-to-lbrn #86)
- pcbburn.com: Robust file-upload loading/error states, live settings, SVG auto-fit, context-menu disable (#17-27)
- tscircuit.com UI: New Builds/Releases pages, GitHub sync button & mobile view, cleaner release details, sidebar tweaks (#2441-2477)
- Footprinter: Convex/concave pad variants, numeric array names (1206x4), mounted-module SVG tests (#470-480)
- Misc: rectdiff stall fix, math-utils helpers, graphics-debug hotkey, status monitor improvements, new SparkFun boards & example KiCad library.
