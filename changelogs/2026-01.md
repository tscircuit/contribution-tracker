# Changelog 2026-01

- Added jumper autorouting with new Pipeline 3 & Curvy Trace solver for tighter, cleaner routing ([core #1821](https://github.com/tscircuit/core/pull/1821), [autorouter #510](https://github.com/tscircuit/tscircuit-autorouter/pull/510), [autorouter #528](https://github.com/tscircuit/tscircuit-autorouter/pull/528))
- PCB Viewer now renders vias, plated holes & SMT pads with hover highlighting plus layer-accurate copper-pour/keepout drawing ([pcb-viewer #590](https://github.com/tscircuit/pcb-viewer/pull/590), [#595](https://github.com/tscircuit/pcb-viewer/pull/595), [#604](https://github.com/tscircuit/pcb-viewer/pull/604))
- circuit-to-canvas: solder-mask margin support for holes & polygons, pcb_hole rotation, and BREP copper-pour rendering ([circuit-to-canvas #104](https://github.com/tscircuit/circuit-to-canvas/pull/104), [#108](https://github.com/tscircuit/circuit-to-canvas/pull/108), [#110](https://github.com/tscircuit/circuit-to-canvas/pull/110), [#112](https://github.com/tscircuit/circuit-to-canvas/pull/112))
- 3D Viewer: STEP→GLB fallback, correct solder-mask clearance for polygonal pads, and oval/pill hole support ([3d-viewer #638](https://github.com/tscircuit/3d-viewer/pull/638), [#651](https://github.com/tscircuit/3d-viewer/pull/651), [#655](https://github.com/tscircuit/3d-viewer/pull/655))
- CLI gains `--ci` flag, pre/post build hooks, richer push summaries, binary clone support & “alwaysUseLatestTscircuitOnCloud” option ([cli #1577](https://github.com/tscircuit/cli/pull/1577), [#1575](https://github.com/tscircuit/cli/pull/1575), [#1568](https://github.com/tscircuit/cli/pull/1568), [#1629](https://github.com/tscircuit/cli/pull/1629), [#1617](https://github.com/tscircuit/cli/pull/1617))
- pcbburn.com: live PCB-burn setting updates, clearer solder-mask controls, instant SVG auto-fit & upload freeze fix ([pcbburn.com #21](https://github.com/tscircuit/pcbburn.com/pull/21), [#22](https://github.com/tscircuit/pcbburn.com/pull/22), [#25](https://github.com/tscircuit/pcbburn.com/pull/25), [#17](https://github.com/tscircuit/pcbburn.com/pull/17))
- circuit-json adds solder-mask offset/size fields & CCW rotation for rectangular pads ([circuit-json #410](https://github.com/tscircuit/circuit-json/pull/410), [#408](https://github.com/tscircuit/circuit-json/pull/408))
- footprinter: convex/concave pad variants and underscore-free resistor-array naming (e.g. “1206x4”) ([footprinter #470](https://github.com/tscircuit/footprinter/pull/470), [#480](https://github.com/tscircuit/footprinter/pull/480))
- easyeda-converter: pin label parser now recognises “#” suffixes ([easyeda-converter #341](https://github.com/tscircuit/easyeda-converter/pull/341))
- 3D panel models now include board cut-outs ([circuit-json-to-gltf #108](https://github.com/tscircuit/circuit-json-to-gltf/pull/108))
- New KicadLibraryConverter API separates user vs. built-in libraries ([circuit-json-to-kicad #47](https://github.com/tscircuit/circuit-json-to-kicad/pull/47))
