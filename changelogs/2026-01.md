# Changelog 2026-01

Key highlights – January 2026

Routing / PCB algorithms
- Autorouter: new jumper-aware Pipeline 3 with AssignablePipelineSolver3, straight-line deviation penalty & keep-out fixes (#510, 506, 511, 521)
- Curvy-Trace Solver integrated and enabled for dense boards; unused jumpers pruned (#528, 529, 524)
- Hypergraph library matured – gradient-descent optimiser, bounds fixes, serialisation, jumper density options (#5, 6, 8, 12, 20)
- Core: jumper autorouting hook-up, internal-connection & port-ID fixes, autorouterVersion & effort level controls exposed (#1821, 1826, 1831, 1833)

Rendering & visualisation
- circuit-to-canvas: solder-mask margin (holes & polygons), copper-pour BREP, PCB-hole rotation, border-radius helper (#104, 108, 110, 112)
- pcb-viewer: via & plated-hole primitive rendering with hover highlighting; keep-out and copper-pour migrated to canvas (#590, 592, 595, 604)
- 3d-viewer: STEP→GLB fallback, pill/oval hole & polygon-pad solder-mask support, type-safety clean-up (#638, 647, 651, 655)
- circuit-to-svg: diagonal keep-out pattern, explicit viewport, <schematicpath/> support (#477, 478, 479)

Tooling & conversion
- CLI: buildCommand / prebuildCommand, ‑-include-dist, ‑-ci, push result summary, global registry URL, KicadLibraryConverter export, cloud “alwaysUseLatestTscircuit” flag, binary-file clone fixes (#1567, 1570, 1565, 1577, 1568, 1579, 1617, 1629)
- circuit-json-to-kicad: new KicadLibraryConverter API & default-export handling (#47, 49)
- easyeda-converter: pin-label “#” suffix parsing & parallel test runner (#341, 342)
- math-utils: point-in-bounds & bounds-center helpers added (#28–31)

Libraries & footprints
- footprinter: convex/concave pads, resistor-array naming (1206x4), mounted-module SVG tests, numeric pad parameters, naming refactor (#470–476, 478–480)
- circuit-json: rectangular-pad solder-mask size props & CCW rotation for circular-pad-with-rect (#408, 410)

Web apps & UI
- pcbburn.com: upload loading/error states, live settings, SVG auto-fit, context-menu disable, validation & width fixes (#17, 18, 19, 20, 21, 23, 25, 26)
- tscircuit.com: new Builds/Releases pages, mobile repo view & manual GitHub sync, sidebar/preview revamp (#2441, 2446–2448, 2470, 2453)
- runframe: retry button & multiple eval/3d-viewer bumps (#2289, 2288, 2295, 2302)

Misc
- rectdiff stall fix for complex polygons (#65)
- status service script & contribution-tracker updates (#58–60, 292, 294)

Version bumps & snapshot updates omitted for brevity.
