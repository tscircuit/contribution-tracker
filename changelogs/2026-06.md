# Changelog 2026-06

- Circuit-JSON/Props/Core
  – Added PCB silkscreen graphic schema (image/BRep) and rectangle corner-radius support (#600, #688, #241).
  – Introduced oval hole interface + rendering support across props, core and converters (#689, #2393).
  – Enabled pcb_copper_text deserialization, pcb_silkscreen_graphic export, dashLength/dashGap on schematic lines/paths (#2362, #570, #2403).
  – Auto-placement for schematic sections, hybrid placement option and improved breakout boundary & label collision handling (#2378, #2390, #2386, #2394, #2396).
  – Normal component footprint resolver + solder-jumper fixes (#2385, #2387).

- Conversion Tooling
  – circuit-json→KiCad: Edge.Cuts keepouts + standalone .kicad_mod / .kicad_sym converters (#330, #134, #126, #136).
  – circuit-json→Gerber: fixed silkscreen knockout rendering (#114).
  – circuit-json→tscircuit: TSX generators for silkscreen, SMT pad, advanced plated holes (#40, #43).
  – circuit-json→canvas/svg: added silkscreen rectangle corner radius & dash support, bumped math-utils (#241, #568).

- EDA Engines & Autorouter
  – Autorouter: direct-overlap via grouping, component detection refactor, new SRJ19/20 BGA datasets (#1336-#1354).
  – Checks: eliminated false “missing connection” errors on branched traces (#158).
  – Calculate-cell-boundaries: multiple repro fixes and cleaner separator removal (#20, #25).

- CLI / Runframe / Eval
  – CLI now uses runtime project config, uploads transitive deps, shows login guidance, and supports footprintLibraryMap override (#3157, #3189, #3182, #3175).
  – Runframe exposes isDoneRendering, better dynamic file paths, frequent bump cycle (#3597, #3622, #3623).
  – Eval adds TS entrypoint (*.ts), browser “exports” mapping, PSPICE fixes and package-browser resolution (#2786, #2798, #2822).

- Ti-Parts-Engine & TI support
  – Browser-compatible TI parts engine with UL bridge, CORS-less proxy, footprint library creation + viewer integration (#10-#25 series).

- Viewer / Visualization
  – 3D-viewer: removed legacy geometry & stubs (#763, #768).
  – CAD component viz: worker-based model parsing for faster load (#18).
  – Pinout SVG download button added to tscircuit.com (#3604).

- Website & Docs
  – Ordering opened to all users; added Create-with-AI footer removal, crisp chat widget, PostHog tracking (#3559, #3557, #3603, #3577).
  – New docs for cornerRadius, keepout zones, project/platform config (#702, #723, #703).

- Miscellaneous
  – PSpice→Spice converter bootstrap, typed fixtures (#1-#4).
  – Matchpack gains fixed component position support (#118, #121).
  – Fabrication operator UI gains live job health dashboard (#14).
  – Several version bumps & package updates across repos to align with new features.
