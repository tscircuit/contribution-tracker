# Changelog 2026-06

Key changes delivered in June 2026

Core libraries & data model
- circuit-json: added PCB silkscreen_graphic schema (+version 0.0.433)
- core:  ‑ automatic schematic-section placement, hybrid placement option  ‑ oval PCB hole support & normal-component footprint resolver  ‑ dashLength / dashGap on schematic lines & paths  ‑ fixed missing / mis-detected net-labels, solder-jumper bridging & inner-label collisions  ‑ routed-trace source-ID assignment, isDoneRendering flag exposed
- checks: removed false “missing-connection” DRC errors for branched traces
- calculate-cell-boundaries: cleaned redundant separators, proportional stroke-width, un-cleaned-line bug-fixes

Import / export / conversion
- kicad-to-circuit-json:   ‑ added .kicad_sch, .kicad_sym & standalone .kicad_mod support   ‑ improved pad-ID deduplication, PCB text sizing, symbol pin-lines
- circuit-json-to-tscircuit: silkscreen shapes, polygon / pill SMT-pad, advanced plated-hole and TSX generation refactor
- circuit-json-to-kicad: Edge.Cuts export of pcb_cutout, version 0.0.148
- circuit-json-to-gerber: fixed silkscreen-knockout rendering

Rendering & viewers
- circuit-to-canvas: PCB silkscreen rect cornerRadius + dash support (v0.0.109)
- 3d-viewer: removed legacy board-geometry & unused stubs
- pcb-viewer / cad-component-viz: math-utils upgrade, worker-based model parsing for faster load

CLI / runframe / eval tooling
- cli v0.1.14xx: runtime project-config, transitive-dep uploads, login guidance, schematic-placement analysis bump
- runframe v0.0.20xx: better dynamic file paths, eval/3d-viewer integrations, isDoneRendering support
- eval: TS entry-points, browser ↔ node entry-map, presupplied-imports fix

Autorouter
- Direct-overlap via grouping for same-net consolidation; new SRJ19 & SRJ20 BGA datasets

Manufacturing & fabrication
- circuit-json-to-bom-csv bumped to 0.0.9
- ngspice-spice-engine: PSpice compatibility mode
- spicets & pspice-to-spice: initial release of typed PSpice parsers / converters

Website (tscircuit.com)
- Pin-out SVG download button, order dialog open to all users
- JLCPCB import message clarified; PostHog tracking for order dialog
- Added Crisp chat widget; preserved hash links while loading; external ‘Create with AI’ footer removed

Miscellaneous

- matchpack: fixed-position support
- ti-parts-engine: browser-compatible, CORS proxy & kicad-converter integration, UL bridge export
- circuit-to-svg: dash support + silkscreen graphic export
- docs: new pages for keepouts, cornerRadius usage, platform config
