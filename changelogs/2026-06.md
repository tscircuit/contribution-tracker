- Core / PCB engine
  – Added oval-hole support and normal-component footprint resolver (#2393, #2385)
  – Auto-placement for un-positioned schematic sections & hybrid placement option (#2378, #2390)
  – Assigned routed-trace source-IDs, fixed missing net-labels and inner-label collisions (#2377, #2379, #2396)
  – Dash length / gap styling for schematic paths & exposed schematic_port data in footprints (#2403, #2405)
  – Solder-jumper and breakout footprint fixes (#2383, #2386)

- Schema & props
  – circuit-json 0.0.433 introduces PCB silkscreen graphic schema (#600)
  – New silkscreen graphic & oval-hole props (#688, #689); removed unsupported cornerRadius on schematics (#686)

- Graphic generators
  – circuit-to-svg: dash-pattern support & pcb_silkscreen_graphic rendering (#568-#570)
  – circuit-to-canvas: PCB silkscreen rect corner-radius drawing (#241)

- Import / export pipelines
  – KiCad → circuit-json gains .kicad_sch / .kicad_sym & standalone .kicad_mod support, schematic_symbol export, duplicate-pad fix, better text sizing (#126, #131, #136, #125, #122)
  – circuit-json-to-KiCad supports pcb_cutout Edge.Cuts export (#330)
  – circuit-json-to-Gerber fixes silkscreen knockout rendering (#114)

- Autorouter
  – Direct-overlap via grouping and new SRJ19/20 BGA datasets (#1334, #1348, #1345)
  – Component detection refactor & routing pipeline re-ordering (#1341, #1343)

- DRC / Checks
  – Eliminated false “missing-connection” errors on branched traces (#158)

- calculate-cell-boundaries
  – Fixed uncleaned lines, proportional stroke-widths, redundant separator cleanup (#20, #22, #25)

- CLI & build tooling
  – Runtime project config across build/export/simulate (#3157)
  – Transitive-dependency upload & login-expiration guidance (#3189, #3182)
  – isDoneRendering integration for faster CI runs (#3221)

- eval / runframe

- TI Parts Engine
