# Changelog 2026-06

Key highlights for June 2026

Core design & rendering
- Silkscreen graphics: new schema in circuit-json, props & SVG/CSS rendering (circuit-json #600, props #688, circuit-to-svg #570, core #2398)
- Dashed schematic lines/paths enabled (circuit-to-svg #568, core #2403)
- Oval/advanced PCB hole and SMT-pad shapes supported end-to-end (props #689, core #2393, cj-to-tscircuit #40 #43)
- PCB cut-outs exported to KiCad Edge.Cuts (cj-to-kicad #330)
- Auto / hybrid schematic placement improvements (core #2378, #2390)
- DRC: fixed false “missing-connection” on branched traces (checks #158)
- Solder-jumper & label collision fixes (core #2387, #2396)

Autorouter & PCB algorithms
- Direct-overlap via grouping for robust same-net consolidation (autorouter #1336)
- Component detection refactor & new BGA benchmark datasets SRJ19/SRJ20 (autorouter #1335, #1345, #1348)

Tooling & runtimes
- CLI now honors runtime project config, uploads transitive deps, adds login guidance (cli #3157, #3189, #3182)
- Runframe gains dynamic file paths + isDoneRendering API; Eval supports .ts entrypoints & browser field mapping (runframe #3597, core #2391, eval #2786, #2798)

Converters & simulation
- Stand-alone KiCad .kicad_mod / .kicad_sym → circuit-json, duplicate-pad fix (kicad-to-cj #126, #131, #125)
- PSPICE → SPICE converter & typed fixtures (pspice-to-spice #1-2, spicets #3-4)
- Ngspice engine now has PSPICE-compat mode (ngspice-spice-engine #10)

Ecosystem libraries
- TI Parts Engine ported to browser, CORS proxy, extensible footprint library (ti-parts-engine #17-25)
- matchpack adds fixed-position chip support (#121)

Viewers & website
- 3D viewer cleanup of legacy geometry (3d-viewer #763 #768)
- Pinout SVG download, ordering for all users, tracking & Crisp chat on tscircuit.com (site #3604, #3559, #3577, #3603)

Bug fixes & maintenance
- calculate-cell-boundaries cleanup (#20 #25) and numerous package bumps/version releases across CLI, Runframe, Eval, and core repositories to keep dependencies current.
