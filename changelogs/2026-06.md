- TI Parts Engine: TypeScript entrypoints, CORS proxy, Ultra-Librarian bridge, browser compatibility.

- Website: pin-out SVG download, order dialog tracking, JLCPCB sign-in clarification, Crisp chat widget added; external footer link removed.
