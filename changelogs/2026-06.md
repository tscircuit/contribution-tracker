# Changelog 2026-06

2026-06 Highlights

Core / Engine
- Added automatic schematic-section placement and hybrid placement support (#2378, #2390)
- Fixed solder-jumper bridging, missing net-label removal, inner-label collisions & power/ground label regex (#2387, #2379, #2396, #2395)
- Introduced normal-component footprint resolver & pcb_copper_text re-hydration (#2385, #2362)

- circuit-json-to-tscircuit: footprint TSX generation over-haul, SMT pad & silkscreen rect/line/circle support, advanced plated-hole shapes (#36, #37, #40, #42-43)
- kicad-to-circuit-json: standalone .kicad_mod & .kicad_sym/.kicad_sch import, duplicate-pad ID fix, improved PCB text & pin-line export (#120-126, #131, #135-136)
- kicadts: exported KiCadSym types & ZoneKeepout constructor (#48-49)

CLI / Runframe / Eval
- CLI now uses runtime project config in all flows, uploads transitive deps, adds login guidance, footprintLibraryMap example, isDoneRendering polling (#3157, #3189, #3182, #3175, #3221)

PCB / Autorouter
