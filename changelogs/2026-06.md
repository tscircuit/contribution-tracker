# Changelog 2026-06

- PCB silkscreen graphics now fully supported – new BRep/image schema, silkscreen props and oval-hole props ([circuit-json #600](https://github.com/tscircuit/circuit-json/pull/600), [props #688](https://github.com/tscircuit/props/pull/688), [props #689](https://github.com/tscircuit/props/pull/689))
- Added corner-radius rendering for PCB silkscreen rectangles ([circuit-to-canvas #241](https://github.com/tscircuit/circuit-to-canvas/pull/241))
- Footprint generator gains silkscreen circle/rect and advanced SMT-pad & hole shapes ([circuit-json-to-tscircuit #40](https://github.com/tscircuit/circuit-json-to-tscircuit/pull/40), [#39](https://github.com/tscircuit/circuit-json-to-tscircuit/pull/39), [#38](https://github.com/tscircuit/circuit-json-to-tscircuit/pull/38), [#37](https://github.com/tscircuit/circuit-json-to-tscircuit/pull/37))
- Automatic schematic-section placement and hybrid placement options introduced ([core #2378](https://github.com/tscircuit/core/pull/2378), [core #2390](https://github.com/tscircuit/core/pull/2390))
- Fixed missing net labels and solder-jumper footprint resolution errors ([core #2379](https://github.com/tscircuit/core/pull/2379), [core #2387](https://github.com/tscircuit/core/pull/2387))
- Routed traces now carry source-port IDs for better trace provenance ([core #2377](https://github.com/tscircuit/core/pull/2377))
- Footprint rehydration adds pcb_copper_text support ([core #2362](https://github.com/tscircuit/core/pull/2362))
- Eliminated false “missing connection” DRC alarms on branched traces ([checks #158](https://github.com/tscircuit/checks/pull/158))
- Cell-boundary calc fixes: stray-line cleanup, proportional stroke width & redundant-separator removal ([calculate-cell-boundaries #20](https://github.com/tscircuit/calculate-cell-boundaries/pull/20), [#22](https://github.com/tscircuit/calculate-cell-boundaries/pull/22), [#25](https://github.com/tscircuit/calculate-cell-boundaries/pull/25))
- tscircuit.com: pinout SVG download button, universal ordering access, preserved package-view hashes, removed “Create with AI” footer ([#3604](https://github.com/tscircuit/tscircuit.com/pull/3604), [#3559](https://github.com/tscircuit/tscircuit.com/pull/3559), [#3558](https://github.com/tscircuit/tscircuit.com/pull/3558), [#3557](https://github.com/tscircuit/tscircuit.com/pull/3557))
- CLI now honors runtime project config, shows login guidance and uploads transitive deps ([cli #3157](https://github.com/tscircuit/cli/pull/3157), [#3182](https://github.com/tscircuit/cli/pull/3182), [#3189](https://github.com/tscircuit/cli/pull/3189))
- Eval resolver: supports `.ts` entrypoints, platformConfig overrides & browser field mapping ([eval #2786](https://github.com/tscircuit/eval/pull/2786), [#2784](https://github.com/tscircuit/eval/pull/2784), [#2798](https://github.com/tscircuit/eval/pull/2798))
- Runframe improves dynamic file-path handling ([runframe #3597](https://github.com/tscircuit/runframe/pull/3597))
- Autorouter consolidates same-net vias via new direct-overlap grouping ([tscircuit-autorouter #1336](https://github.com/tscircuit/tscircuit-autorouter/pull/1336))
- Matchpack now supports fixed component positions ([matchpack #121](https://github.com/tscircuit/matchpack/pull/121))
- TI Parts Engine: browser compatibility and `createTiFootprintLibrary` helper added ([ti-parts-engine #17](https://github.com/tscircuit/ti-parts-engine/pull/17), [#22](https://github.com/tscircuit/ti-parts-engine/pull/22))
- KiCad importer: supports `.kicad_sym` files plus improved symbol scaling/spacing & pin-line output ([kicad-to-circuit-json #119](https://github.com/tscircuit/kicad-to-circuit-json/pull/119), [#120](https://github.com/tscircuit/kicad-to-circuit-json/pull/120), [#121](https://github.com/tscircuit/kicad-to-circuit-json/pull/121))
- New PSpice tooling: typed fixtures & ngspice ABM serialization in Spicets, plus pspice-to-spice converter ([spicets #4](https://github.com/tscircuit/spicets/pull/4), [pspice-to-spice #1](https://github.com/tscircuit/pspice-to-spice/pull/1))
