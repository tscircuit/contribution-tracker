# Changelog 2026-05

- Rectangular reroute regions now supported in autorouting phases, with shape/tolerance props and auto-inserted default phase ([core #2249](https://github.com/tscircuit/core/pull/2249), [props #659](https://github.com/tscircuit/props/pull/659), [core #2260](https://github.com/tscircuit/core/pull/2260))
- New autorouting-phase component flow and `unrouteCircuitJson` helper enable clean re-route workflows ([core #2236](https://github.com/tscircuit/core/pull/2236), [core #2245](https://github.com/tscircuit/core/pull/2245))
- Integrated KRT WebAssembly autorouter preset for faster routing on all platforms ([tscircuit #3148](https://github.com/tscircuit/tscircuit/pull/3148), [eval #2596](https://github.com/tscircuit/eval/pull/2596))
- KiCad `.kicad_pcb` files can now be imported directly as Board components ([eval #2574](https://github.com/tscircuit/eval/pull/2574))
- Major KiCad import upgrades: KiCad-10 support, Edge.Cuts circles, copper pours, text rotation, via graphing & JLCPCB part props preserved ([kicad-to-circuit-json #74](https://github.com/tscircuit/kicad-to-circuit-json/pull/74), [#72](https://github.com/tscircuit/kicad-to-circuit-json/pull/72), [#89](https://github.com/tscircuit/kicad-to-circuit-json/pull/89), [#87](https://github.com/tscircuit/kicad-to-circuit-json/pull/87), [#86](https://github.com/tscircuit/kicad-to-circuit-json/pull/86), [#71](https://github.com/tscircuit/kicad-to-circuit-json/pull/71))
- circuit-json-to-kicad now exports copper pours and fixes rotated pill plated-hole footprints ([circuit-json-to-kicad #264](https://github.com/tscircuit/circuit-json-to-kicad/pull/264), [#294](https://github.com/tscircuit/circuit-json-to-kicad/pull/294))
- Gerber generator extended to 8-layer boards and correct soldermask/pad handling for vias, plated holes & polygon SMT pads ([circuit-json-to-gerber #85](https://github.com/tscircuit/circuit-json-to-gerber/pull/85), [#93](https://github.com/tscircuit/circuit-json-to-gerber/pull/93), [#95](https://github.com/tscircuit/circuit-json-to-gerber/pull/95), [#91](https://github.com/tscircuit/circuit-json-to-gerber/pull/91))
- DRC engine adds polygon-aware rotated-pad checks, better missing-trace error placement and hole-courtyard overlap detection ([checks #143](https://github.com/tscircuit/checks/pull/143), [#145](https://github.com/tscircuit/checks/pull/145), [#147](https://github.com/tscircuit/checks/pull/147))
- Schematic props improvements: optional port name/direction, per-net nominalTraceWidth and JLCPCB footprint autocomplete ([props #658](https://github.com/tscircuit/props/pull/658), [props #657](https://github.com/tscircuit/props/pull/657), [props #669](https://github.com/tscircuit/props/pull/669), [props #671](https://github.com/tscircuit/props/pull/671))
- Fixed PCB via & trace inflation issues, preserving original layer and clearances in reroutes ([core #2278](https://github.com/tscircuit/core/pull/2278), [core #2251](https://github.com/tscircuit/core/pull/2251))
