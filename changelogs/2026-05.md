# Changelog 2026-05

- Autorouter: rectangular region rerouting supported via shape=rect ([core #2249](https://github.com/tscircuit/core/pull/2249), [props #659](https://github.com/tscircuit/props/pull/659))
- Autorouter: new "krt" WASM preset selectable in platform config ([props #661](https://github.com/tscircuit/props/pull/661), [eval #2596](https://github.com/tscircuit/eval/pull/2596))
- AutoroutingPhase now accepts custom DRC parameters & auto-inserts a default phase when phaseIndex is omitted ([core #2267](https://github.com/tscircuit/core/pull/2267), [core #2260](https://github.com/tscircuit/core/pull/2260))
- DRC: polygon-aware checks for rotated pads & plated holes added ([checks #143](https://github.com/tscircuit/checks/pull/143))
- DRC: mounting holes inside their own courtyard now correctly flagged ([checks #149](https://github.com/tscircuit/checks/pull/149))
- Gerber export: 8-layer boards now supported ([circuit-json-to-gerber #85](https://github.com/tscircuit/circuit-json-to-gerber/pull/85))
- Gerber export: correct solder-mask margins for SMT pads & plated holes ([circuit-json-to-gerber #95](https://github.com/tscircuit/circuit-json-to-gerber/pull/95), [#98](https://github.com/tscircuit/circuit-json-to-gerber/pull/98))
- Import: `.kicad_pcb` files can be brought in as Board components ([eval #2574](https://github.com/tscircuit/eval/pull/2574))
- CLI: static file loading added (enables offline `.kicad_pcb` imports) ([cli #2987](https://github.com/tscircuit/cli/pull/2987))
- KiCad → Circuit JSON: knockout text preserved on import ([kicad-to-circuit-json #87](https://github.com/tscircuit/kicad-to-circuit-json/pull/87))
- Viewer: new “Bounds” tool lets you drag-select a rectangular view area ([pcb-viewer #864](https://github.com/tscircuit/pcb-viewer/pull/864))
- UI: order dialog CSS isolated to avoid clashing with host pages ([order-dialog #1](https://github.com/tscircuit/order-dialog/pull/1))
