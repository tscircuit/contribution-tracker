# Changelog 2026-05

Key highlights for May 2026

- Autorouting
  – Introduced “autorouting phases” with rectangular & reroute-only modes (core #2236, 2249, 2260, 2267)
  – Added krt-wasm preset & pipeline solver refactor (core #2256, props #661, autorouter #1140, 1148)
  – Via-/pad-edge clearance solver & region-reroute gap-fill improvements (autorouter #1113, 1123, 1137; rectdiff #128-131)

- KiCad round-trip & file import
  – Full KiCad 10.0 support incl. zones, vias inside pcb_trace and board snapshot parity (kicad-to-circuit-json #74, 80-86)
  – Import .kicad_pcb directly as Board component & static-file loading in CLI/Runframe (eval #2574, 2595; cli #2987)
  – Circuit-json-to-KiCad: rotated pill plated-hole & drill-colour fixes (cj-to-kicad #294, 297)

- DRC & manufacturing checks
  – Polygon-aware DRC for rotated pads / plated holes (checks #143)
  – New via-pad, hole-courtyard and missing-trace error positioning (checks #144-149)
  – Gerber: blind-via drill, 8-layer output & copper/paste mismatch fixes (cj-to-gerber #85-89)

- Schematic & layout features
  – Group schematic box pin placement using port direction + collision solvers (core #2237, schematic-placement-analysis #18-19)
  – Support through-pad route type & preserved imported pcb_trace elements (circuit-json #579; core #2262)
  – Footprint inflation, via inflation and courtyard layer generation fixes (core #2251, 2278; cj-to-tscircuit #30)

- Props / component data
  – Footprint shorthand mapping & JLCPCB autocomplete (props #662, 664, 671)
  – New fields: shape for autorouting regions, nominalTraceWidth, optional port name/direction, Via layers array (props #657-660, 668-669)

- Tooling & viewers
  – Added unrouteCircuitJson helper (core #2245)
  – Circuit-to-SVG handles through-pad segments & text-anchor fix (svg #560-561)
  – PCB-viewer rectangle-bounds selection tool (pcb-viewer #864)

- Documentation & examples
  – Docs for autorouting phases, schematic box rendering, local STEP model example (docs #549-553)
  – New SparkFun boards, Arduino-UNO & thermocouple breakout test datasets (sparkfun-boards #267-282, core #2269)
