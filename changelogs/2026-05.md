# Changelog 2026-05

May 2026 – Key changes across the TSCircuit ecosystem

- Autorouting phases
  – core #2236, #2249, #2260, #2267: first-class "autoroutingPhase" component, rectangular/reroute support, default phase insertion, per-phase DRC params.
  – props #659, #660, #661: shape, tolerance & krt-wasm preset fields added.

- Region & reroute reliability
  – core #2256, #2296, #2295: platform-supplied router, correct source-trace IDs, Arduino-Uno region cleanup.
  – autorouter #1119, #1137, #1146: reroutable SRJ dataset, clipped-boundary handling, failing-reachability snapshot.

- Kicad 10 integration round-trip
  – kicad-to-circuit-json #74-#86: KiCad v10 parser, zones & vias inside pcb_trace, connectivity graph, Edge.Cuts gr_circle, JLCPCB props preservation.
  – eval #2574, #2595: import kicad_pcb as Board component + boardContentCircuitJson export.
  – cli #2987: static file loading for kicad_pcb.

- PCB geometry accuracy
  – core #2278-#2279: via inflation overhaul.
  – circuit-json-to-kicad #297; c-json-to-gerber #89: rotated pill plated hole & blind-via drill fixes.
  – checks #147, #149: courtyard vs. holes/footprints overlap detection.

- DRC & benchmarking
  – checks #143, #144, #145: polygon-aware pad checks, multi-pad port false-positive fix, accurate missing-trace coordinates.
  – high-density-repair03 #9-#17: DRC14 benchmark harness, parallel execution, interactive debugger.

- Footprint & schematic quality
  – props #662-#665, #668: resistor footprint autocomplete/mapping, via layers schema, resistance type fix.
  – circuit-json-schematic-placement-analysis #18-#19: label collision & diode-resistor alignment solvers.
  – schematic-trace-solver #286-#289: trace/label overlap fixes.

- New utilities & APIs
  – core #2245: unrouteCircuitJson helper.
  – circuit-json new "through_pad" route type (#579).
  – circuit-to-svg #560-#561: through-pad segments & knockout-text anchor rendering.

- 3D / visualization
  – c-json-to-gltf #160: hide fallback boxes for empty footprints.
  – pcb-viewer #864: rectangle-drag “Bounds” tool.

- Miscellaneous enhancements
  – order-dialog #1 CSS scoping; props footprintFileParser refactor (#663).
  – sparkfun-boards new MAX31855K & RJ11 breakouts (#281, #282).
