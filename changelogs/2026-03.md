- PCB-Viewer: hide group overlay on subpanels, decluttering the view — [pcb-viewer #687](https://github.com/tscircuit/pcb-viewer/pull/687)
- circuit-json: 
  - new `simple_connector` spec — [circuit-json #487](https://github.com/tscircuit/circuit-json/pull/487)

- Core panelization/layout:

  - stop subpanels from triggering group layout; include nested boards in panelization — [#2005](https://github.com/tscircuit/core/pull/2005)
- Core error handling improvements:
  - emit `pcb_missing_footprint_error` when a chip lacks a footprint — [#1998](https://github.com/tscircuit/core/pull/1998)
  - clearer message for invalid `pinLabel` errors — [#1997](https://github.com/tscircuit/core/pull/1997)

  - concurrent GLB conversion step — [#2233](https://github.com/tscircuit/cli/pull/2233)

  - PCB output enabled for placement checks & new `routingDisabled` flag — [#2210](https://github.com/tscircuit/cli/pull/2210)
