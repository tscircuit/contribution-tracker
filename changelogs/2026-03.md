# Changelog 2026-03

Key highlights for 2026-03

- Sub-panels: viewer no longer shows group overlay (#687) and core now skips group layout, supports grid layout & nested boards in panelization (#2005, #2002).
- DRC flexibility: optional per-platform DRC-disable flags added in props/core/CLI (#608, #2003, #2017) plus global disable flag (#611). Connector-orientation check introduced (#107) and bulky pad-overlap check removed (#105).
- Connector initiative: simple_connector spec in circuit-json (#487); core adds <connector /> element and auto cable-insert center (#2011); warning for inaccessible orientations (#491).
- Placement API: pcb_component anchors now optional with new “relative_to_another_component” mode (#485).
- Visibility & notes: pcbSx.visibility + fabricationnotetext support (props/core #609, #610, #2016); toggle pcb_note_* rendering in 3D and 2D viewers (3d-viewer #721, circuit-to-canvas #202).
- Courtyards: courtyard elements excluded from outside-board DRC (#2014); SVG/Docs/rendering & tscircuit conversion libs gain courtyard support (svg.tscircuit.com #1140, docs #491, c-json-to-tscircuit #25).
- Error clarity: new pcb_missing_footprint_error (#1998) and better invalid pinLabel messaging (#1997); improved PCB-overlap errors (#106).
- Autorouter gains nominal trace-width support (#2019) and big speed/quality boosts via baked via-tiles & solver optimisations (hypergraph #112/123/125/126, autorouter #619/621/626).
- CLI leaps: pseudo-orthographic camera, binary upload in local sync, GLB conversion & snapshot concurrency, build prop injection, netlist checking/JSON search, improved init defaults & libraryEntrypoint, worker pattern, and strict DRC-error exit (cli #2232, #2229, #2233, #2239, #2277-#2287, #2269, #2263, #2255, #2275).
- 3D-viewer overhaul: trace/via/SMT-pad textures now via circuit-to-canvas; per-layer trace splitting, plated-hole masking, better panel/board detection & CAD fallback (3d-viewer #723-#730).
- PCB canvas engine: copper drill cut-outs & trace-endpoint clipping (#204, #206).
- New footprints: JST ZH connector support (#514) plus SOT-223-5 & TO-220F-3 (#515).
- Utilities & fixes: getBoardBounds helper (#81) & bounds bug-fix (#83); KiCad mod-cache 404 resolved (#14); schematic auto-layout for custom symbols (#2015); trace render crash guard (#2012); implicit single-pin selectors & Component.pin shorthand (#2010, #2007).
- Analysis tooling: placement-analysis package adds pin-to-pin distance & nearest-clearance outputs (#1-#5).
- Documentation: new pages for <connector/>, courtyard elements, and updated CLI/skill guides (#492, #491, skill #7-#10).
- Many version bumps/pack-updates across repos keep dependencies current.
