# Changelog 2026-03

- Core
  – Added connector element, implicit single-pin selectors & Component.pin shorthand (#2011, #2010, #2007)
  – Sub-panel aware grid/panelization layouts & group-overlay suppression (#2005, #2002)
  – Platform-level DRC-disable flags and nominal trace-width support for autorouter (#2003, #2017, #2019)
  – Better error handling: missing footprint, invalid pinLabel, unresolved port, overlap msgs (#1998, #1997, #2012, checks#106)
  – Exclude courtyards / fab notes from outside-board DRC bounds; visibility/pcbSx support (#2014, #2016)

- Circuit-JSON / Props / Util
  – simple_connector spec, inward-facing connector warning, cable_insertion_center field (#487, #491, #489)
  – Optional pcb_component anchors + relative_to_another_component positioning (#485)
  – PlatformConfig: per-check & global DRC disable flags, rename visible→visibility (#608, #611)
  – getBoardBounds helper & plated-hole/offset bounds fix (util#81, #83)

- CLI
  – New camera presets (pseudo-orthographic, snapshots) & GLB conversion concurrency (#2232, #2187, #2233)
  – Binary upload in local sync, stallTimeout, --inject-props, libraryEntrypoint support (#2229, #2235, #2239, #2263)
  – Netlist workflow: tsci check netlist (human & JSON), routing disabled option (#2279, #2281, #2282)
  – Concurrency flag for snapshots, improved tsci init defaults, DRC-exit-code fix (#2277, #2269, #2275)

- 3D / PCB & SVG Viewers
  – Refactored texture generation (traces, SMT/via/panel outlines) to circuit-to-canvas (#730-726, #724-723)
  – Toggle pcb_notes_* via context menu; correct CAD Z-placement; showCourtyards passthrough (#721, #729, svg#1140)
  – pcb-viewer: hide group overlay for subpanels (#687)

- Autorouter / Hypergraph
  – Faster solvers: Flatbush obstacle checks, baked via-tiles, automatic via-tile selection (#619, hypergraph#112, #121-125)
  – Solver tuning (MAX_ITERATIONS, cost factors, dataset03 fixture) (#621, hypergraph#117)

- Checks
  – Removed obsolete footprint-pad overlap check (#105) and moved connectivity checks to routing (#110)
  – New connector accessibility orientation check & repro tests (#107, #115)

- Footprinting & Fabrication
  – JST-ZH, SOT-223-5, TO-220F-3 footprints added (#514, #515)
  – Drill/trace & solder-mask rendering fixes in circuit-to-canvas (#206, #204)

- Misc
  – KiCad-mod-cache 404 fix (#14); better search in jlcsearch (#136)
  – Extensive docs updates: connector/courtyard pages, package build API, skill guides (#491-493, docs#489, skill#5-10)
