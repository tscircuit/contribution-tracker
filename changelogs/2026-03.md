# Changelog 2026-03

- Core / DRC
  • core#2005 – Sub-panels no longer trigger group layouts; nested boards now panelize correctly.
  • core#2003, props#608, core#2017 – Added platform-level drcChecksDisabled & per-check flags.
  • core#2014 – Exclude Courtyard / Fabrication notes from outside-board DRC.
  • core#1998 – Emit pcb_missing_footprint_error when footprint is absent.
  • core#2011 – New <connector /> element with cable-insert-center auto-guess + related Circuit-JSON warnings (cjson#491).
  • core#2010, #2007 – Simplified trace selectors: implicit single-pin and Component.pin shorthand.
  • checks#107 – New connector accessibility orientation check; checks#106 – clearer overlap errors; checks#105 – removed self-pad-overlap check.

- Layout / Autorouting
  • core#2019 – Nominal trace-width support feeds autorouter.
  • tscircuit-autorouter#619, #621, #626 – Faster obstacle checks via Flatbush, tuned solver constants, new profiling scripts.
  • hypergraph#112, #119-126 – Automatic via-tile selection, baked via-tile topology; big speed & determinism gains.

- Viewer / Rendering
  • pcb-viewer#687 – Hide group overlay inside sub-panels.
  • 3d-viewer#721, #724, #726-730 – Major texture renderer refactor: per-layer trace segments, drilled-hole masks, accurate SMT/through-hole Z-positions, context menu toggles for pcb_notes.
  • circuit-to-canvas#202-207 – Added showPcbNotes toggle, copper cut-outs for drills/vias, solder-mask improvements.

- CLI / Tooling
  • cli#2232 – New pseudo-orthographic camera preset; direction normalizer.
  • cli#2229 – Local-sync uploads now support multipart/form-data for binaries.
  • cli#2239 – tsci build gains --inject-props/--inject-props-file.
  • cli#2277, #2280 – --concurrency flag for snapshot; faster automation.
  • cli#2279, #2281 – `tsci check netlist` with readable output, routing disabled by default.
  • cli#2263 – Support libraryEntrypoint in config; improved `tsci init` defaults (#2269).

- File-format / Libraries
  • circuit-json#485, #489 – Optional anchors, relative_to_another_component placement, cable_insertion_center.
  • circuit-json#487 – Introduced simple_connector spec.
  • circuit-json-to-tscircuit#22, #25 – Automatic schematic-symbol & courtyard generation from Circuit-JSON.
  • footprinter#514, #515 – Added JST-ZH, SOT-223-5 & TO-220F-3 footprints.
  • props#609-612 – pcbSx.visibility, fabricationnotetext selector, plated-hole shape inference.

- Bug fixes
  • core#2012 – Prevent PCB trace render crash on unresolved port selectors.
  • core#1997 – Clearer invalid pinLabel errors.
  • kicad-mod-cache#14 – Resolve 404s when loading cached KiCad footprints.
  • circuit-json-util#83 – Correct board-bounds calc affecting plated-hole offsets.

- Documentation & API
  • docs#491-493 – Added courtyard element pages, <connector /> docs, package-builds API.
  • skill#7-10 – Updated best-practice guides for connector usage, netlist checks and new CLI features.
