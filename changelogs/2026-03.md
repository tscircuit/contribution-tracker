# Changelog 2026-03

- Core
  – Added full sub-panel support: grid packing (#2002) and no unintended group layouts (#2005)
  – New connector model incl. auto cable-insert center guess (#2011) and implicit single-pin trace selectors (#2010)
  – Autorouter: nominal trace-width support (#2019)
  – Better error handling: missing footprint error (#1998), clearer pin-label & overlap messages (#1997, checks#106)
  – Crash fixes for unresolved port selectors & custom-symbol auto-layout (#2012, #2015)

- Circuit-JSON / Props
  – simple_connector spec + inward-facing-connector warning (#487, #491)
  – Optional anchors, relative-placement mode & cable_insertion_center fields (#485, #489)
  – Platform-wide DRC disable flags + global “disableAll” switch (#608, #610, #611)
  – pcbSx.visibility & fabricationnotetext selector support (#609, core#2016)
  – Auto plated-hole shape inference (#612)

- Checks
  – New connector-accessibility placement check (#107)
  – PCB trace connectivity moved to routing stage (#110)
  – Removed own-pad-overlap check (#105)

- CLI (tsci)
  – Camera presets incl. pseudo-orthographic & 3D snapshot options (#2232, #2187)
  – Faster builds: concurrent GLB conversion & binary upload via multipart (#2233, #2229)
  – New workflow commands: tsci check netlist (human & JSON output), search --json, --inject-props, --concurrency flags (#2279, #2282, #2239, #2277)
  – Library entrypoint support & improved init defaults (#2263, #2269)
  – Client-server build worker pattern (#2255) and option to disable routing during netlist check (#2281)

- 3D / PCB Viewers
  – Toggle pcb_note* visibility in 3D (#721) and PCB overlay hidden for sub-panels (pcb-viewer#687)
  – Major texture pipeline refactor: traces split per layer, plated holes/vias & SMT pads use Circuit-to-Canvas drawer (#724, #726, #727, #730)
  – Better panel outline detection (#723) and fallback CAD model URL chain (#728); fixed Z-positioning of CAD parts (#729)

- Rendering (circuit-to-canvas / svg)
  – Copper & mask rendering fixes: drill holes cut from copper, trace ends trimmed at drills, showPcbNotes toggle (#204, #206, #202)
  – getBoardBounds util added to circuit-json-util (#81)

- Autorouter / Hypergraph
  – Obstacle checks accelerated with Flatbush and via-tile baking for deterministic topology (autorouter#619, hypergraph#121-126)
  – Tuned cost constants, added profiling & benchmark fixtures (#621, #626, #629)

- Footprinter / Libraries
  – New footprints: JST-ZH, SOT-223-5, TO-220F-3 with hyphenated alias support (#514, #515)
  – KiCad mod-cache 404 fix (#14)

- Misc
  – Contribution-tracker star & sponsorship threshold updates (#318-319)
  – LBRN exporter gains separate top/bottom soldermask layers (#149)
  – GLTF exporter fixes browser WASM loading (#131)
