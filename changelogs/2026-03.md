# Changelog 2026-03

Key enhancements delivered in March 2026
- Core
  – Added <connector/> element, automatic cable-insertion center & implicit single-pin selectors (#2011, 2010)
  – Full sub-panel support (grid layout, no group overlay, nested boards) (#2005, 2002; pcb-viewer #687)
  – Platform-level DRC disable flags & visibility overrides (#2003, 2017; props #608, 611)
  – New errors & clearer messages: missing footprint, invalid pinLabel, trace render guard (#1998, 1997, 2012)
  – Nominal trace-width + async footprint-port extraction (#2019, 2008)

- CLI (tsci)
  – Pseudo-orthographic/3D camera presets & snapshot concurrency (#2232, 2277)
  – Local sync now supports multipart binary upload (#2229)
  – New “tsci check netlist” & JSON search output; routing can be disabled (#2279, 2281, 2282)
  – GLB conversion parallelism, libraryEntrypoint, improved init defaults (#2233, 2263, 2269)
  – Builds now fail on unresolved DRC errors (#2275)

- Autorouter / Hypergraph
  – Obstacle checking optimised with Flatbush, higher iteration caps (#619, 621, 626)
  – Via-tile baking & automatic selection → deterministic, faster topology (#112, 121-126)

- 3D / Rendering
  – Trace, pad, via & panel-outline textures migrated to circuit-to-canvas pipeline (#726-727, 730, 723)
  – Alternate CAD model fallback chain (OBJ/WRL/STL→GLTF→STEP) (#728)

- Circuit-to-Canvas / SVG
  – Drill/via copper cut-outs & trace-endpoint clipping; pcb_notes visibility toggle (#204-207)
  – SVG now exposes showCourtyards flag (svg.tscircuit.com #1140; docs #493)

- Circuit JSON & Utils
  – simple_connector spec, optional anchors, relative positioning, cable_insertion_center (#487, 485, 489)
  – Warning for inward-facing connectors (#491); new getBoardBounds util (#81)

- Checks
  – New connector-accessibility check, clearer overlap/trace messages; removed pad-overlap check (#107, 106, 105)
  – Trace-connectivity checks moved to routing stage (#110)

- Props / Footprinter
  – DRC-disable & plated-hole inference support; renamed pcbSx.visible → visibility (#608-612)
  – Added JST ZH, SOT-223-5 & TO-220F footprints (#514-515)

- Other notable fixes & docs
  – Duplicate build log suppression (tscircuit.com #2922)
  – KiCad mod-cache 404s resolved (#14)
  – Contribution-tracker star-score & sponsorship threshold updates (#318-319)
  – Better JLC PCB search relevance (#136)

(Version-bump and routine update PRs omitted for brevity.)
