# Changelog 2026-03

Key enhancements delivered in March 2026

Design-time & DRC
- core: Full sub-panel support – omit group overlay / layout fixes (#2005, #2002, pcb-viewer #687)
- core/props: Allow platform-wide DRC-disable flags & per-object visibility toggle (#2003, props #608, #610, #611)

- circuit-json: new simple_connector spec, inward-facing-connector warning & cable_insertion_center, plus relative-anchor positioning (#487, #491, #489, #485)

Authoring CLI
- New camera presets incl. pseudo-orthographic; concurrency for snapshots & GLB conversion (#2232, #2187, #2233, #2277)
- Multipart binary upload for local sync (#2229)
- New build flags: --inject-props / file, libraryEntrypoint, stallTimeout, global concurrency option (#2239, #2263, #2235, #2277)
- Netlist workflow: tsci check netlist (readable output, routing-disabled) and search --json (#2279, #2281, #2282)

Rendering & Visualization
- 3d-viewer: major texture refactor – plated holes, vias, traces, SMT pads; panel outline detection; CAD Z-fix; toggle pcb_notes; fallback CAD URLs (#724, #726-#730, #723, #721, #728, #729)
- circuit-to-canvas: drill-hole copper cut-outs + showPcbNotes toggle (#204-#206, #202)
- svg.tscircuit.com: show_courtyards flag passthrough (#1140)

Routing & Placement Engines
- autorouter/hypergraph: Flatbush-powered obstacle checks, baked via-tiles & automatic via-tile selection for faster deterministic topology (#619, #112, #121-#125, #626)
- core: nominal trace-width support & connector element with auto cable-insert center (#2019, #2011)

- kicad-mod-cache: fixed 404s when loading footprints (#14)

Tooling & Docs

Minor but notable
