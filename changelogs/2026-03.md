# Changelog 2026-03

- Better sub-panel support: hide group overlays, include nested boards in panelization & grid layouts ([pcb-viewer #687](https://github.com/tscircuit/pcb-viewer/pull/687), [core #2005](https://github.com/tscircuit/core/pull/2005), [core #2002](https://github.com/tscircuit/core/pull/2002))
- New connector workflow: `<connector />` element with auto cable-insert center, inward-facing-orientation warning & related JSON spec updates ([core #2011](https://github.com/tscircuit/core/pull/2011), [circuit-json #487](https://github.com/tscircuit/circuit-json/pull/487), [circuit-json #489](https://github.com/tscircuit/circuit-json/pull/489), [circuit-json #491](https://github.com/tscircuit/circuit-json/pull/491), [checks #107](https://github.com/tscircuit/checks/pull/107))
- Platform-wide DRC overrides: per-flag and global disable options in props & core ([props #608](https://github.com/tscircuit/props/pull/608), [props #611](https://github.com/tscircuit/props/pull/611), [core #2017](https://github.com/tscircuit/core/pull/2017))
- Autorouter now honours nominal trace width setting ([core #2019](https://github.com/tscircuit/core/pull/2019))
- Friendlier diagnostics: explicit missing-footprint error, clearer pin-label & overlap messages, ambiguous-port warnings instead of crashes ([core #1998](https://github.com/tscircuit/core/pull/1998), [core #1997](https://github.com/tscircuit/core/pull/1997), [core #2022](https://github.com/tscircuit/core/pull/2022), [checks #106](https://github.com/tscircuit/checks/pull/106))
- CLI upgrades:
  • Pseudo-orthographic & additional 3-D camera presets ([cli #2232](https://github.com/tscircuit/cli/pull/2232), [cli #2187](https://github.com/tscircuit/cli/pull/2187))
  • `tsci build` supports `--inject-props{,-file}` ([cli #2239](https://github.com/tscircuit/cli/pull/2239))
  • New `tsci check netlist` command w/ readable output ([cli #2279](https://github.com/tscircuit/cli/pull/2279))
  • Search now has `--json` output & multi-word queries ([cli #2282](https://github.com/tscircuit/cli/pull/2282), [cli #2309](https://github.com/tscircuit/cli/pull/2309))
  • `tsci snapshot` & GLB conversion accept `--concurrency` ([cli #2277](https://github.com/tscircuit/cli/pull/2277))
  • One-pass preview generation flags (`--pngs`, `--svgs`, etc.) ([cli #2304](https://github.com/tscircuit/cli/pull/2304))
  • Smarter `tsci init` defaults ([cli #2269](https://github.com/tscircuit/cli/pull/2269))
- 3-D/Canvas rendering improvements:
  • Context-menu toggle for PCB note layers (off by default) ([3d-viewer #721](https://github.com/tscircuit/3d-viewer/pull/721))
  • Per-layer trace textures for accurate PCB visuals ([3d-viewer #727](https://github.com/tscircuit/3d-viewer/pull/727))
  • Auto-fallback to alternate CAD model formats before error ([3d-viewer #728](https://github.com/tscircuit/3d-viewer/pull/728))
  • Correct CAD component Z-position on boards & in panels ([3d-viewer #729](https://github.com/tscircuit/3d-viewer/pull/729), [core #2026](https://github.com/tscircuit/core/pull/2026))
  • Copper & mask rendering fixes: drill/via hole cut-outs & trace-end trimming ([circuit-to-canvas #204](https://github.com/tscircuit/circuit-to-canvas/pull/204), [circuit-to-canvas #206](https://github.com/tscircuit/circuit-to-canvas/pull/206))
- Footprinter & JSCAD: new JST ZH, SOT-223-5, TO-220F-3 footprints and pill-shaped plated-hole support ([footprinter #514](https://github.com/tscircuit/footprinter/pull/514), [footprinter #515](https://github.com/tscircuit/footprinter/pull/515), [jscad-electronics #281](https://github.com/tscircuit/jscad-electronics/pull/281))
