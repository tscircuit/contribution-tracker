# Changelog 2026-03

Key changes delivered in 2026-03
- Connector workflow
  – Added simple_connector spec + inward-facing-connector warning (circuit-json #487 #491)
  – core now auto-guesses cable insertion centre, supports Connector.pin shorthand & emits better missing-footprint/pin-label errors (core #2011 #1998 #1997)
  – New DRC check for connector accessibility (checks #107)

- Sub-panel & panelization
  – Sub-panels no longer show overlay in viewer (pcb-viewer #687)
  – Grid/packed layouts and nested-board panelization supported, avoiding unwanted group layouts (core #2005 #2002)

- DRC / platform controls
  – Platform-level drcChecksDisabled flags exposed in props & consumed by core/CLI, including “disable all” option (props #608 #611 | core #2003 #2017)
  – Removed obsolete footprint-pad-overlap check (checks #105) and improved overlap messages (checks #106)

- Autorouter & routing
  – Flatbush-powered obstacle checks + solver tuning boost speed (autorouter #619 #626 #621)
  – Nominal trace-width support added (core #2019)

- Rendering & viewers
  – 3D-viewer texture pipeline refactor: traces, pads, plated holes, panel outlines; trace split per layer; CAD fall-back URLs; Z-offset fix; context menu toggles for pcb_notes_* (3d-viewer #721 #723 #724 #726-#730)
  – circuit-to-canvas drills-in-copper masking + trace endpoint trimming; showPcbNotes toggle (c-to-canvas #204 #206 #202)
  – SVG & docs gain showCourtyards support (svg.tsc #1140, docs #491)

- CLI improvements (v0.1.1026 → v0.1.1060)
  – Pseudo-orthographic/3D camera presets, concurrency flags, stallTimeout, binary-file uploads, inject-props, libraryEntrypoint, improved init defaults (cli #2232 #2233 #2235 #2229 #2239 #2263 #2269)
  – New commands: tsci check netlist (readable output) & --json output for search; routing disabled for placement checks (cli #2279 #2282 #2281)
  – Build no longer exits on benign DRC in circuit-json (cli #2275)

- Data & analysis tooling
  – New circuit-json-placement-analysis pkg with pin-to-pin clearance reporting (cj-pa #1-#5)
  – Board-bounds util (cj-util #81) and plated-hole offset fix (#83)

- Performance & algorithms
  – Hypergraph: baked via-tile topology + automatic tile selection (hypergraph #112 #123-#126)

- Footprints & libraries
  – Added JST ZH, SOT-223-5, TO-220F-3 footprints (footprinter #514 #515)
  – Default plated-hole shape inference (props #612)

- Miscellaneous
  – Better KiCad mod-cache 404 handling (#14), mask-layer fix for LBRN export (cj-to-lbrn #149), improved contribution-tracker star/sponsor thresholds (#317-#319), docs & examples for courtyard/connector elements (#492)
