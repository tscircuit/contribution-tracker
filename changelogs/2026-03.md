# Changelog 2026-03

- Circuit-JSON schema: simple_connector & cable_insertion_center fields, inward-facing connector warning, optional anchors/relative positioning, getBoardBounds util
- Props & Core: platformConfig flags to disable specific/all DRC checks; pcbSx.visibility + fabricationnotetext support; clearer pin-label / missing-footprint errors
- Checks: new connector-orientation placement check, richer overlap messages; moved trace-connectivity checks into routing stage; removed obsolete pad-overlap check
- Core PCB engine: full sub-panel support (grid + panelization), implicit single-pin trace selectors, Component.pin shorthand, async footprint-port extraction, trace-width hint for autorouter, crash fixes
- New <connector/> element & automatic cable-insert-center guessing; courtyard/notes excluded from outside-board DRC
- Viewer stack: pcb-viewer hides group overlay for sub-panels; 3D-viewer refactors textures (traces, SMT pads, vias, outlines), fixes CAD Z-placement & adds pcb_notes toggle; circuit-to-canvas adds drill-hole cutting & notes visibility
- CLI (tsci): pseudo-orthographic camera preset, snapshot/3D camera presets, multipart binary upload, glb-conversion concurrency, --inject-props flag, netlist check with readable/JSON output, --concurrency for snapshot, improved init defaults, libraryEntrypoint, stallTimeout & client-server worker mode; search --json output
- Autorouter/HG: Flatbush-accelerated obstacle checks, baked via-tiles & automatic tile selection (dataset improvements), tunable solver constants, nominal trace-width support
- Footprinter: JST-ZH, SOT-223-5, TO-220F-3 footprints with alias support
- New utilities & conversions: dedicated solder-mask layers in LBRN export; courtyard outline/rect support in circuit-json→tscircuit; STEP→GLTF demo & WASM fixes in GLTF converter
- Contribution-tracker: updated star/crown thresholds & sponsorship docs
- Documentation: new pages for connector, courtyard elements, Package Builds API, skill guides on CLI checks/commands
